# 阶段 V · 债务与待补

本文件记录阶段 V 在“主章展开完成”之后，已经稳定的部分和仍需继续深化的部分。

## 一、已经闭合的部分

### 1.1 路线已经清楚

当前阶段 V 的路线已经固定为：

- `§41` 边界切换
- `§42` 实例不变性
- `§43` 基础视觉特征
- `§44` 结构化密集表征
- `§45` 预测式表征
- `§46` 图像世界状态
- `§47` 反身度量

这条线已经可以稳定承接阶段 IV 的出口残差。

### 1.2 本体对象已经压清

阶段 V 当前不再模糊地谈“更强理解”，而明确以：

- 对象
- 部件
- 几何
- 遮挡
- 对应
- 结构稳定性
- 布局关系
- 状态不确定性

作为图像世界状态的收敛对象。

### 1.2.1 核心语法已经补入

本轮已新增 [01-阶段V核心语法-状态等变预测反身](01-阶段V核心语法-状态等变预测反身.md)，把阶段 V 的纯存在态压成：

```text
x -> z_S
z_S(x) = evidence-constrained partial state
```

并加入 `H0-H5` 状态持有分级、不变性/等变性边界、III/IV/V 边界表和阶段 V 七次否定。

### 1.2.2 前沿补遗与状态对象谱系已经补入

本轮新增：

- [02-阶段V前沿补遗-2024到2026-04-从好表征到状态对象](02-阶段V前沿补遗-2024到2026-04-从好表征到状态对象.md)
- [03-状态对象谱系-阶段V到底持有什么](03-状态对象谱系-阶段V到底持有什么.md)

阶段 V 的前沿口径因此从“好表征”进一步推进为“状态对象”：每条新方法都要说明自己支撑的是语言无关表征、dense feature、预测状态、单图几何、多视图几何、持续 3D/4D 状态，还是反身评测。

### 1.3 `§47` 已经把终极残差推出台面

阶段 V 现在不再只是“自监督方法楼层”，而是已经明确触到了：

- 度量冲突
- probe 合法性
- 世界状态应如何被评估

这使它真正接上了 `反身闭合`。

### 1.4 一手前沿证据已经补齐到主线级

本轮并行研究之后，阶段 V 不再只靠概念判断支撑，而已经有一手材料回灌到主章：

- JEPA / I-JEPA / V-JEPA 线，支撑 `§45`
- DINO / DINOv2 / DINOv3 与 dense 自蒸馏线，支撑 `§43`
- Language-free SSL scaling、RADIO / C-RADIO、Perception Encoder、SigLIP2，支撑 `§43` 的前沿补遗
- iBOT / DenseCL / PixPro / DetCon / LOST / TokenCut / STEGO，支撑 `§44-§46`
- Depth Pro / Depth Anything V2 / Metric3D / UniDepth / DUSt3R / MASt3R / VGGT / Matrix3D / CUT3R / D4RT，支撑 `§46` 的几何与 4D 状态补遗
- IntPhys2 / CausalVQA，支撑 `§47` 的负证据页
- VTAB / disentanglement evaluation / representation similarity，支撑 `§47`

正式索引入口见 [95-并行研究证据索引](95-并行研究证据索引.md)。

## 二、仍然存在的债务

### 2.1 文献注引已经立起骨架，但还未完全饱和

当前阶段 V 已补上章节级“一手参考”与阶段级总表 [96-阶段V核心参考文献](96-阶段V核心参考文献.md)，这意味着参考体系已经从无到有正式立起。
但它还没有完全饱和，后续仍要继续补：

- 更细粒度的章节内行文锚点
- 更完整的显式会议 / 期刊信息
- 更稳定的 frontier 更新机制

### 2.2 `00-定位卡` 仍可进一步 agent 化

现在定位卡足够用，但尚未补充：

- 核心公式索引
- 三问协议
- 读取深度建议

### 2.3 `§46` 还可继续向具体任务证据压实

目前 `§46` 已经把世界状态对象定义清楚，并新增 [§46/02-图像世界状态对象登记表](§46-图像中的世界状态持有/02-图像世界状态对象登记表.md)。未来仍可继续往下压到更细的证据链，如：

- 深度一致性
- 遮挡推理
- correspondence
- part consistency
- 状态不确定性与多假设预测

### 2.4 `§47` 仍是整座楼里最开放的一章

这是正常现象。因为一旦写到反身度量，问题就开始超出单篇方法论，进入体系层审计。它目前已经足够作为终点章，但后续最可能继续生长的也是这一章。

本轮新增 [§47/02-阶段V证据强度与反Probe协议](§47-反身度量与终极残差/02-阶段V证据强度与反Probe协议.md)，把证据分成 `E0-E6`。这已经能防止把 linear probe、attention map、dense transfer、state prediction 混成同等强度的证明，但仍需要继续补：

- `E4-E5` 级任务证据；
- 反例库；
- 多度量排序反转案例；
- `Rfl-M` 模型反身的真实机制。

### 2.5 新增概念债务

本轮加硬之后，新的债务更清楚：

- 不变性/等变性边界需要更多任务例证。
- `H0-H5` 每一级需要代表方法、反例和失败样本。
- 单图状态欠定性需要和遮挡、amodal、counterfactual benchmark 对接。
- 几何状态、物理状态、因果状态之间的边界需要持续补反例。
- `State Object Audit` 需要进一步制度化成每篇前沿论文的登记卡。
- 阶段 V 的生成性预测需要每半年回到横切登记簿验证。

### 2.6 研究线笔记还没有完全制度化

目前并行研究材料已经产生，但还属于“阶段内证据索引 + 专题笔记”状态。若后续继续扩大 frontier 研究，最好形成统一协议：

- 哪些属于正式主章内容
- 哪些属于阶段内证据附录
- 哪些应上升到横切层长期维护

## 三、当前建议

如果继续推进，最正确的顺序是：

1. 先保持阶段 V 当前主线不大改。
2. 后续主要补“注引”和“证据制度化”，而不是再改章节次序。
3. 进入横切层，把 `反身脊柱`、`预测性判据登记簿` 和证据强度分级与阶段 V 对齐得更紧。
4. 继续把 `H0-H5` 与 `E0-E6` 做成可维护的审计工具，而不是只停留在本轮文档。

## 四、当前结论

阶段 V 现在已经可以视为：

> 一层主命题明确、主章可读、对象收敛清楚、并且真正接上终极残差的正式楼层。

它还远没到“终稿”，但已经足够承接整座大厦的后半程。
