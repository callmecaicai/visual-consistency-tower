# 阶段 V · §44 机制展开：表征如何开始持有对象、部件与密集结构

类型: 理论/主章  
阶段: V · 表征共形  
共形维度: 材料, 世界表征, 度量  
关键贡献: 说明为什么 dense / patch / object-centric 表征是阶段 V 的必要中层。没有这一步，世界状态只能停留在“一个很强的全局 embedding”这种模糊说法里。  
残差 / 它催生了什么: 结构虽然开始被显化，但系统仍主要停留在“描述已有结构”而非“预测隐藏或未来状态”。这条残差直接送向 `§45` 的 JEPA 主线。  
定位标签: dense feature, patch token, iBOT, object semantics, dense representation

## 学习抓手

- 本章的重点不是再证明 backbone 很强，而是证明“全局向量不够”。
- 第一次读本章，先抓住一句话：世界状态如果不能落到对象、部件、patch 与对应关系上，就仍然只是空泛的全局表示。
- 本章要解决的是“显化结构”，不是“再刷高一个分类分数”。

## 一、为什么全局向量不足以承载世界状态

到了 `§43`，视觉已经有了很强的通用特征。但一个总问题仍然悬着：

> 如果表征最终只压成一个全局向量，那么对象、部件、边界、局部关系和空间布局到底存在哪里？

全局 embedding 很适合：

- 分类
- 检索
- 粗粒度相似性

但它不天然适合：

- 对象部件
- 密集对应
- 空间边界
- 遮挡与局部关系

因此阶段 V 必须进一步推进：从“有强特征”转向“有结构化特征”。

## 二、patch 级表征为什么关键

视觉不是纯句向量问题。图像里的世界状态天然带有空间分布：

- 左右上下关系
- 部件归属
- 局部纹理与整体轮廓的耦合
- 前后遮挡和边界过渡

这意味着如果表征要真正开始持有世界，最小单位不能只是整张图，而必须下沉到 patch、token、区域和对象层级。

也就是说：

> 世界状态不是“整个图的一句话总结”，而是一组分布在空间中的局部-整体组织关系。

## 三、iBOT 与 patch-level 自蒸馏

iBOT 的重要性之一，就在于它不再只在全局视图上做一致性，而开始让 patch 级别的 token 也参与到自蒸馏结构里。

这带来两个关键后果：

1. 表征不再只是“整图是同一个”。
2. 表征开始在局部单位上学习稳定结构。

于是视觉表征第一次显得更像一个内部场，而不是一个压缩标签向量。

这一步对阶段 V 很重要，因为它开始让“对象和部件”有了具体承载位置。

## 四、从 emergent structure 到 direct shaping

如果只看 DINO，很容易把这一层误读成：结构只是偶然从强表征里“冒出来”。但一手结果已经表明，后来的 dense SSL 路线并不满足于等它自然涌现，而是开始直接把局部结构写进目标函数。

这里至少有三类关键证据：

1. **DenseCL**：直接把 local correspondence 放进目标函数，说明局部对应不是 probe 幻觉，而是可被直接塑形的训练对象。
2. **PixPro**：同时要求 `spatial sensitivity` 与 `spatial smoothness`，这恰好对应边界、区域内部一致性和部件组织。
3. **DetCon**：把对齐从 patch 推到 object-level features across augmentations，说明“对象作为稳定实体”可以进入预训练目标。

这三条线一起说明：

> 结构化表征不是只会从 backbone 里自然冒出来，它也可以被直接训练出来。

## 五、对象语义如何从 dense 结构中浮现

对象不是凭空从标签表里掉下来的。对于阶段 V 来说，对象语义的真正出现方式更像这样：

- 某些 patch 在多视角、多增强下总是共同变化。
- 某些局部边界总是在结构上稳定。
- 某些部件总在同一个更大单元内共同出现。

于是“对象”不再只是类别名，而开始表现为：

> 一簇在空间中共同变化、共同稳定、可被同一结构解释的局部状态。

这就把对象从外部标签重新写回了内部表征几何。

但这里必须区分三类对象：

| 对象层级 | 含义 | 典型证据 | 边界 |
|---|---|---|---|
| 视觉对象 | 由边界、纹理、颜色、共变 patch 形成的 perceptual object | foreground emergence、TokenCut、LOST | 不能直接等同于物理实体 |
| 语义对象 | 能被类别、名称、语言或任务标注识别的 object | segmentation、classification、grounding | 不能直接等同于视觉自持状态 |
| 物理对象 | 在遮挡、视角变化、时间变化中保持身份和空间连续性的 entity | correspondence、occlusion、multi-view consistency | 不能只靠单图 objectness 宣称完成 |

因此，本章的 objectness emergence 主要说明视觉对象和部分语义对象开始显形，还不能直接证明物理对象已经被持有。物理对象层需要 `§46` 的遮挡、视角、对应和不确定性证据。

## 六、边界、前景与对象可分离性为什么重要

如果世界状态真的存在于图像表征里，那么它至少应该支持：

- 前景与背景的可分离；
- 对象边界的稳定切开；
- 对象在无标签条件下仍能被发现；
- 区域之间的语义相关性。

`LOST`、`TokenCut` 与 `STEGO` 的价值就在这里。它们说明：

- 自监督 ViT 的特征几何里确实存在强 objectness 信号；
- 无标签对象发现、边界切开和区域语义分区可以从这些信号中读出；
- 但 object extraction 仍依赖图切、聚类或其他算法制度，并不等于模型内部已经显式持有物理对象实体。

因此，本章不是在说“dense 很有用”，而是在说：

> 只有当边界、区域和对象可被稳定切开，世界状态才开始从全局 embedding 中脱壳。

## 七、dense structure 为什么是世界状态的必要前提

如果世界状态要在图像里被持有，至少要能承载这些东西：

- 哪些局部属于同一对象
- 哪些局部是前景、哪些是背景
- 哪些边界是结构性边界
- 哪些区域在另一张图里与之对应

这些都不是单个全局向量能优雅表达的。它们需要：

- 稠密表征
- 空间对齐
- 局部-整体一致性

因此，本章的结论不是“dense feature 很有用”，而是：

> **没有 dense / structured representation，就没有阶段 V 意义上的世界状态。**

## 八、这一层仍然还不够

虽然对象、部件、patch 结构开始显化，但阶段 V 到这里仍然更多是在：

- 描述
- 对齐
- 保持结构一致

它还没有真正走到更深一步：

> 不只是持有已经可见的结构，而是直接预测隐藏、缺失或尚未出现的抽象状态。

换句话说，本章解决的是“世界状态长什么样”，但还没解决“世界状态如何被预测”。

## 九、本章真正要记住什么

- 强 backbone 不等于强世界状态持有，因为全局 embedding 太粗。
- patch、对象、部件、dense structure 是世界状态开始显形的必要中层。
- iBOT 及相关 dense 自蒸馏路线的重要性，在于让结构下沉到局部 token 层。
- DenseCL、PixPro、DetCon 说明 correspondence、边界和平滑区域可以被直接写入训练目标，而不只是事后读出。
- LOST、TokenCut、STEGO 则说明对象性和边界可分离性已经在特征几何中获得了直接证据。
- 对象语义在本章里不再主要来自标签，而来自共同变化和共同稳定的局部组织。

## 十、向 §45 的传递

到这里，阶段 V 的问题会自然升级为：

> 如果这些结构已经能被持有，那么系统能否不靠像素重建，而直接去预测抽象状态本身？

这就是 `§45` 的 JEPA 主线。

## 十一、本章一手参考

- [Caron et al., *DINO* (2021)](https://arxiv.org/abs/2104.14294)  
  证明对象边界与前景显形会从 SSL ViT 中 emergent 地冒出来。

- [Zhou et al., *iBOT* (2021/2022)](https://arxiv.org/abs/2111.07832)  
  让结构下沉到 patch token 层，是本章最关键的 patch-level 自蒸馏来源。

- [Wang et al., *DenseCL* (2020/2021)](https://arxiv.org/abs/2011.09157)  
  说明 local correspondence 可以被直接写成训练目标。

- [Xie et al., *PixPro* (2020/2021)](https://arxiv.org/abs/2011.10043)  
  支撑边界敏感性、区域内部平滑性与 dense consistency。

- [Hénaff et al., *DetCon* (2021)](https://arxiv.org/abs/2103.10957)  
  说明 object-level 对齐不是下游读出，而可以成为预训练目标。

- [Siméoni et al., *LOST* (2021)](https://arxiv.org/abs/2109.14279)、[Wang et al., *TokenCut* (2022)](https://arxiv.org/abs/2209.00383)、[Hamilton et al., *STEGO* (2022)](https://arxiv.org/abs/2203.08414)  
  共同支撑“对象性、边界可分性、区域语义一致性”已经存在于特征几何内部。
