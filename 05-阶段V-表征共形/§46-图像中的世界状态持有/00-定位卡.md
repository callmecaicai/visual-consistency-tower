# 阶段 V · §46 定位卡：图像中的世界状态持有

类型: 理论/定位卡
阶段: V · 表征共形
共形维度: 世界表征, 度量, 主体性
关键贡献: 把“世界表征”在当前写作范围中收敛定义为：对象、部件、几何、深度、遮挡、对应、布局与状态不确定性在图像表征中的受证据约束持有。
残差 / 它催生了什么: 一旦世界状态被具体化，阶段 V 的最后问题就不再是“表征是什么”，而是“什么度量能合法评判这些表征”，这送向 `§47`。
定位标签: geometry, depth, correspondence, occlusion, world state

## 本章位置

这一章是整个阶段 V 的收敛落点。它把“真正的世界表征”从空泛大词压回到图像视觉里的可讨论对象。

核心边界是：图像世界状态不是完整世界真值，而是 `evidence-constrained partial state`。正式登记表见 [02-图像世界状态对象登记表](02-图像世界状态对象登记表.md)。

## 代表线索

- depth / geometry consistency
- correspondence / matching
- occlusion / layout / part-whole structure
- uncertainty / multi-hypothesis state
- multiview image consistency 但仍停留在图像证据，不展开视频 rollout

## 向下一章的传递

当世界状态已经被压回图像结构本身，最后的问题就变成：什么才算一个合法的评价标准？这送向 `§47`。
