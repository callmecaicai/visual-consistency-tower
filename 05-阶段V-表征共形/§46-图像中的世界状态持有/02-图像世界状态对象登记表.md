# 阶段 V · §46 补页：图像世界状态对象登记表

类型: 状态登记/协议
阶段: V · 表征共形
共形维度: 世界表征, 度量, 证据
关键贡献: 把“图像中的世界状态”拆成可审计的状态对象，防止它滑成泛泛的 world model 叙事。
残差 / 它催生了什么: 状态对象一旦登记，下一步就必须说明哪些证据有资格证明状态被持有，这送向 `§47` 的证据强度与反 Probe 协议。
定位标签: state registry, object identity, occlusion, correspondence, uncertainty

## 一、登记原则

图像世界状态不是完整世界真值，而是：

```text
S(x) = {s | O(s) is compatible with x}
```

也就是说，图像给出的不是全知世界，而是一组受观测约束的可能状态。阶段 V 的“持有”只能在这个范围内成立。

登记状态对象时必须回答五个问题：

1. 它持有的是什么。
2. 它有哪些可观察代理。
3. 它能通过什么接口被调用。
4. 它在哪些变换下应该稳定或等变。
5. 它有哪些典型失败模式。

## 二、状态对象总表

| 状态对象 | 它持有的是什么 | 可观察代理 | 典型接口 | 失败模式 |
|---|---|---|---|---|
| 对象身份 | 同一实体在变化下仍为同一 | instance correspondence、tracking-like consistency、clustering | retrieval、matching、segmentation | 同物异名、异物同名 |
| 部件拓扑 | part-whole 结构与局部归属 | part segmentation、amodal part consistency、object parsing | part parsing、editing、fine-grained recognition | 部件漂移、结构拼错 |
| 几何深度 | 3D 布局、深度、法向、尺度 | depth、normal、pose、view consistency | 3D、depth、pose、view synthesis | 语义强但几何弱 |
| 遮挡关系 | 谁在前、谁被遮住、不可见部分如何延续 | amodal completion、occlusion reasoning、layer ordering | completion、scene reasoning、editing | 可见处对，遮挡处乱 |
| 跨图对应 | 不同图中同一局部、表面或对象 | dense correspondence、matching、semantic correspondence | transfer、editing、tracking | 局部纹理误配 |
| 布局关系 | 支撑、相邻、包含、上下左右、接触 | relation prediction、scene graph、layout consistency | VQA、grounding、planning-like query | 关系词对但空间错 |
| 状态不确定性 | 哪些状态被证据固定，哪些只是可能 | uncertainty、multi-hypothesis prediction、counterfactual consistency | occlusion、counterfactual、amodal reasoning | 过度确定、幻觉补全 |

## 三、三类“对象”不能混用

阶段 V 中的 object 至少有三层：

| 对象层级 | 含义 | 典型证据 | 不能偷换成什么 |
|---|---|---|---|
| 视觉对象 | 由边界、纹理、颜色、共变 patch 形成的 perceptual object | TokenCut、LOST、foreground emergence | 不能直接等同于物理实体 |
| 语义对象 | 能被类别、名称、语言或任务标注识别的 object | segmentation、classification、grounding | 不能直接等同于视觉自持状态 |
| 物理对象 | 在遮挡、视角变化、时间变化中保持身份和空间连续性的 entity | correspondence、occlusion、multi-view / tracking-like consistency | 不能只靠单图 objectness 宣称完成 |

对象边界显现是强证据，但它通常仍停留在视觉对象或部分语义对象层。要进入物理对象层，必须补上遮挡、视角、对应和时间延续证据。

## 四、单图状态的欠定性

单张图像天然欠定：

- 被遮挡物体的完整形状不唯一。
- 深度和尺度可能有多解。
- 背面结构不可见。
- 图片外延展区域不可知。
- 因果意图和未来运动不能由单图直接确定。

因此，成熟的状态表征不应只给一个确定 latent，还应表达：

```text
fixed by evidence
possible under evidence
unknown under current evidence
```

阶段 V 的强表征不是“把不知道的都补成确定答案”，而是能区分哪些状态被图像证据固定，哪些只能以不确定性形式持有。

## 六、本页纪律

后续任何方法若声称“进入阶段 V”，必须落到本页至少一个状态对象上。不能只说：

```text
feature 更好
probe 更高
transfer 更强
attention map 更漂亮
```

必须说：

```text
它持有哪类状态？
状态在哪些变换下稳定或等变？
通过什么接口调用？
哪些证据支持？
哪些反例会使它破裂？
```
