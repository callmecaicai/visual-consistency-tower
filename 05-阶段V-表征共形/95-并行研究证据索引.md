# 阶段 V · 并行研究证据索引

本文件收束本轮多 agent 并行探索的正式结论，服务两类用途：

- 给人类读者：快速看清阶段 V 当前最硬的一手证据从哪里来。
- 给 agent：在不通读全部主章的前提下，快速定位 `§43-§47` 的前沿支撑材料。

## 一、这轮并行研究最后确认了什么

阶段 V 当前最稳的判断，可以压成五条：

1. `JEPA` 不是 masked modeling 的小变体，而是把学习对象从“观测恢复”切换到“抽象状态预测”。
2. `DINO -> DINOv2 -> DINOv3` 这条线说明，视觉表征确实会长成可迁移、可 dense transfer、可支撑对象边界与 correspondence 的基础设施。
3. `DenseCL / PixPro / iBOT / DetCon / LOST / TokenCut / STEGO` 这条线说明，图像里的世界状态并不是空词，它至少会落到对象、边界、部件、对应、几何等可操作对象上。
4. 从机构路线看，`Meta FAIR` 最明确地把“非生成式预测表征 / state prediction / think-before-act”推成主线；`DeepMind / NVIDIA / OpenAI` 更强地走向生成式世界模拟；两者不能混写。
5. `VTAB`、Google 的 disentanglement 评估、BAIR 的表征相似性分析说明：一旦阶段 V 以表征为终点，问题就必然升级为度量冲突，而不再只是“哪个 backbone 更强”。

用途：

- `IntPhys 2` 与 `CausalVQA` 说明：状态预测成立，不等于世界理解已经闭合。

### 2.2 研究线 B：DINO / DINOv2 / DINOv3 与视觉基础特征

- [§43/研究线2-一手来源梳理](§43-自蒸馏与视觉基础特征/研究线2-一手来源梳理.md)

当前最硬结论：

- `Registers` 与 `DINOv3` 说明：dense feature 并不会天然纯净，阶段 V 的“基础特征”不是静态答案，而是持续被校正的底座工程。

- 本轮结论已回灌到 [00-总纲](00-总纲-从图像样本到世界状态的持有.md)、[§45](§45-JEPA与预测式表征/01-机制展开.md)、[§46](§46-图像中的世界状态持有/01-机制展开.md)、[§47](§47-反身度量与终极残差/01-机制展开.md)

- `Microsoft / OpenAI`：更接近统一多模态接口、VLM、agent 与生成系统。

## 三、这些证据最后怎样改写阶段 V

如果只看主章，阶段 V 会显得像一条理论链；加上这轮并行证据以后，它已经变成一条有前沿支撑的判断链：

1. `§41`：重建不是终点，而是转向状态预测前的桥。
2. `§42`：实例不变性是最小起点，但还远不是世界状态。
3. `§43`：视觉表征开始成为可蒸馏、可迁移、可 dense transfer 的基础设施。
4. `§44`：对象、部件、边界、correspondence 与 dense structure 被具体化。
5. `§45`：JEPA 把学习对象从观测恢复切到抽象状态预测。
6. `§46`：图像世界状态被压回对象、部件、几何、遮挡、对应、布局。
7. `§47`：一旦这些状态成为主对象，系统就必须回答“什么度量才合法”。
