# 阶段 V 前沿补遗：从好表征到状态对象

类型: 前沿补遗 / 状态对象索引
阶段: V · 表征共形
核心命题: 2024-2026 的阶段 V 前沿不再只是“有没有好表征”，而是“哪一种状态对象已经被视觉系统稳定持有”。
残差 / 它催生了什么: dense feature、预测式表征、几何 foundation model、4D scene state 都在扩大阶段 V 的证据面，但它们仍不能自动推出物理、因果与反身度量闭合。

## 一、为什么要补这一页

阶段 V 的原主线已经清楚：

```text
重建边界 -> 样本不变性 -> 基础特征 -> dense structure -> JEPA -> 图像世界状态 -> 反身度量
```

但 2024-2026 的前沿把问题推得更尖锐：

> 表征不再只是 feature。表征开始成为状态对象的容器。

因此，阶段 V 的阅读方式要从“方法谱系”升级为“状态对象谱系”。不要只问 DINOv3、V-JEPA 2.1、Depth Pro、VGGT、D4RT 分别是什么，而要问：

```text
它们让视觉系统持有什么状态？
这些状态到了 H0-H5 的哪一级？
用什么证据裁判？
留下什么残差？
```

## 四、六条线的理论位置

### V-A：纯视觉自持线

这条线回答阶段 III 留下的残差：视觉是否必须永远借语言获得语义稳定性？

Language-free SSL scaling 的价值不在于“打败 CLIP”这个口号，而在于它把同一数据条件下的视觉自监督 scaling 推到台面。它说明纯视觉表征路线仍有独立扩张空间。

边界也必须写清：它最多直接支撑 H1-H3。它不能自动证明模型持有 H4 世界状态。

### V-B：dense feature 线

DINOv3 的关键不只是分数更高，而是把 dense feature quality 作为 scaling 的中心问题。Registers、Gram anchoring 和长训稳定性都在说明：dense token field 不是自然干净的对象，必须被制度化地稳定。

### V-C：聚合表征线

RADIO / C-RADIO、Perception Encoder、SigLIP2 的意义是：单一训练目标不再足以覆盖所有接口，多个教师、多个层级、多个对齐目标被压进一个 encoder。它们形成的是多闭合制度的聚合表征，而不是单一世界状态的完成。

### V-D：几何状态线

Depth Pro、Depth Anything V2、Metric3D、UniDepth、DUSt3R、MASt3R、VGGT、Matrix3D 把阶段 V 的状态对象从“好特征”压实到 depth、normal、camera、pose、pointmap、track。几何开始成为 foundation-level state object。

但几何不是物理。几何强不等于 causal state 闭合。

### V-E：3D / 4D 状态线

CUT3R、D4RT、V-JEPA 2.1 把状态持有推进到流式观察、persistent state、4D reconstruction 与 action-facing representation。它们是阶段 V 向视频和世界模型扩张的证据，但不替代阶段 V 的核心定义：图像证据约束下的部分状态持有。

### V-F：反身评测线

IntPhys2 和 CausalVQA 是必要的负证据。它们提醒我们：即使 dense feature、geometry、tracking、planning interface 都在进步，物理持续性、反事实、预期和因果仍未闭合。

阶段 V 的终极残差因此不是“模型还不够大”，而是：

> 我们仍需要说明什么证据才足以证明某类状态被持有。

## 五、阶段 V 的最新总句

> 阶段 V 的前沿不再是“有没有好表征”，而是“哪一种状态对象已经被视觉系统稳定持有”。

更短地说：

```text
feature -> state object -> audit
```

阶段 V 的下一个工作，不是继续把所有新论文塞进自监督谱系，而是给每个新论文登记：

```text
状态对象是什么？
证据等级是多少？
持有级别到哪里？
它遮蔽了什么？
它如何把阶段 V 推向反身度量？
```
