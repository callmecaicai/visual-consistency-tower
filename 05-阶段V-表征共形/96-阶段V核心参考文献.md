# 阶段 V · 核心参考文献

本文件是 `05-阶段V-表征共形` 的正式参考入口。目标不是穷尽全领域，而是给阶段 V 当前主线提供一套足够硬、可复核、可长期维护的一手来源。

使用原则：

- 只优先收原始论文、正式会议页、OpenReview、CVF Open Access、机构官方研究页。
- 每条参考都服务于阶段 V 的一个明确命题，不收纯背景性泛综述。
- 章节主文档只挂“本章一手参考”，本文件负责给出阶段级总表。

## 一、IV -> V 的边界桥梁：重建为什么不再是终点

1. [He et al., *Masked Autoencoders Are Scalable Vision Learners* (arXiv 2021)](https://arxiv.org/abs/2111.06377)  
   用途：`§41` 的桥梁主文。证明重建式目标可以学出强表征，但对象仍然是可见观测。

2. [Bao et al., *BEiT: BERT Pre-Training of Image Transformers* (arXiv 2021)](https://arxiv.org/abs/2106.08254)  
   用途：`§41` 的离散 token 重建路线。说明“恢复观测”可以换成视觉 token 恢复，但仍未切到状态预测。

3. [Xie et al., *SimMIM: A Simple Framework for Masked Image Modeling* (arXiv 2021)](https://arxiv.org/abs/2111.09886)  
   用途：`§41` 的简化像素回归路线。说明重建桥梁并不依赖复杂 tokenizer。

4. [Zhou et al., *iBOT: Image BERT Pre-Training with Online Tokenizer* (arXiv 2021 / ICLR 2022)](https://arxiv.org/abs/2111.07832)  
   用途：同时连接 `§41` 与 `§44`。它兼具 masked prediction 与 patch-level self-distillation 两面性，是 IV/V 之间最典型的过渡方法之一。

## 二、最小正式起点：实例不变性

5. [Wu et al., *Unsupervised Feature Learning via Non-Parametric Instance-level Discrimination* (arXiv 2018)](https://arxiv.org/abs/1805.01978)  
   用途：`§42` 的历史起点。把“每个实例都是自己的类”明确写成目标。

6. [He et al., *Momentum Contrast for Unsupervised Visual Representation Learning* (arXiv 2019)](https://arxiv.org/abs/1911.05722)  
   用途：`§42` 的系统化工程节点。用动量编码器和队列把实例不变性扩成主流路线。

7. [Chen et al., *A Simple Framework for Contrastive Learning of Visual Representations* (SimCLR, ICML 2020)](https://arxiv.org/abs/2002.05709)  
   用途：`§42` 的核心语法来源。把 InfoNCE、增强策略与大 batch 依赖写清。

## 三、基础视觉特征：表征如何变成基础设施

8. [Grill et al., *Bootstrap Your Own Latent* (arXiv 2020)](https://arxiv.org/abs/2006.07733)  
   用途：`§43` 的去负样本节点。证明不显式拉开负样本也能学出强表征。

9. [Caron et al., *Unsupervised Learning of Visual Features by Contrasting Cluster Assignments* (SwAV, NeurIPS 2020)](https://arxiv.org/abs/2006.09882)  
   用途：`§43` 的在线聚类节点。让表征从实例对比转向可组织的中层空间。

10. [Caron et al., *Emerging Properties in Self-Supervised Vision Transformers* (DINO, ICCV 2021)](https://arxiv.org/abs/2104.14294)  
    用途：`§43` 和 `§44` 的关键文献。证明自监督 ViT 中会出现对象边界与前景显形。

11. [Oquab et al., *DINOv2: Learning Robust Visual Features without Supervision* (arXiv 2023)](https://arxiv.org/abs/2304.07193)  
    用途：`§43` 的平台化节点。把表征推成 `all-purpose visual features`。

12. [Darcet et al., *Vision Transformers Need Registers* (arXiv 2023)](https://arxiv.org/abs/2309.16588)  
    用途：`§43` 的纠偏文献。说明 dense token 场并不天然干净。

13. [Siméoni et al., *DINOv3* (arXiv 2025)](https://arxiv.org/abs/2508.10104)  
    用途：`§43` 的最新延长线。把 high-quality dense features 与长训稳定性推到前台。

## 四、dense structure 与对象语义：世界状态如何从全局 embedding 中脱壳

14. [Wang et al., *Dense Contrastive Learning for Self-Supervised Visual Pre-Training* (arXiv 2020 / CVPR 2021)](https://arxiv.org/abs/2011.09157)  
    用途：`§44` 和 `§46`。用 local correspondence 直接塑形 dense representation。

15. [Xie et al., *Propagate Yourself: Exploring Pixel-Level Consistency for Unsupervised Visual Representation Learning* (arXiv 2020 / CVPR 2021)](https://arxiv.org/abs/2011.10043)  
    用途：`§44` 和 `§46`。把边界敏感性与区域平滑性明确写进目标函数。

16. [Hénaff et al., *Efficient Visual Pretraining with Contrastive Detection* (arXiv 2021)](https://arxiv.org/abs/2103.10957)  
    用途：`§44`。说明 object-level 对齐可以成为自监督目标。

17. [Zhou et al., *iBOT* (arXiv 2021 / ICLR 2022)](https://arxiv.org/abs/2111.07832)  
    用途：`§44`。让结构下沉到 patch token 层。

18. [Siméoni et al., *Localizing Objects with Self-Supervised Transformers and no Labels* (LOST, arXiv 2021)](https://arxiv.org/abs/2109.14279)  
    用途：`§44`。说明无标签对象发现已经潜伏在 SSL ViT 的特征几何里。

19. [Wang et al., *TokenCut: Segmenting Objects in Images and Videos with Self-supervised Transformer and Normalized Cut* (arXiv 2022)](https://arxiv.org/abs/2209.00383)  
    用途：`§44`。进一步支撑对象边界与前景分离。

20. [Hamilton et al., *Unsupervised Semantic Segmentation by Distilling Feature Correspondences* (STEGO, arXiv 2022)](https://arxiv.org/abs/2203.08414)  
    用途：`§44`。说明 dense feature correlation 已经带有 semantic consistency。

## 五、本体切换：为什么状态预测不是像素重建变体

21. [LeCun, *A Path Towards Autonomous Machine Intelligence* (OpenReview 2022)](https://openreview.net/pdf?id=BZ5a1r-kVsf)  
    用途：`00-总纲`、`§45`、`§47`。阶段 V 最重要的总纲性文本之一，明确提出 predictive world model / JEPA 路线。

22. [Assran et al., *Self-Supervised Learning from Images with a Joint-Embedding Predictive Architecture* (CVPR 2023)](https://arxiv.org/abs/2301.08243)  
    用途：`§45` 的主文。I-JEPA 是图像线最干净的状态预测实例。

23. [Bardes et al., *Revisiting Feature Prediction for Learning Visual Representations from Video* (V-JEPA, arXiv 2024)](https://arxiv.org/abs/2404.08471)  
    用途：`§45` 的外推接口。说明 feature prediction 可单独支撑视频视觉学习。

24. [Meta, *Our New Model Helps AI Think Before it Acts* (official post, 2025-06-11)](https://about.fb.com/news/2025/06/our-new-model-helps-ai-think-before-it-acts/)  
    用途：`§45`、`§46`。机构级地把 `understanding, prediction, planning` 明确挂到 V-JEPA 2 上。

25. [Fan et al., *Scaling Language-Free Visual Representation Learning* (arXiv 2025)](https://arxiv.org/abs/2504.01017)  
    用途：`§45`。说明纯视觉表征路线本身可以继续逼近强基础模型，而不必先借语言中介。

26. [Assran et al., *V-JEPA 2: Self-Supervised Video Models Enable Understanding, Prediction and Planning* (arXiv 2025)](https://arxiv.org/abs/2506.09985)  
    用途：`§45` 的外推上限与 `§46` 的边界提醒。

27. [Bordes et al., *IntPhys 2* (arXiv 2025)](https://arxiv.org/abs/2506.09849)  
    用途：`§45`、`§47`。负证据，说明 world reasoning 仍未闭合。

28. [Foss et al., *CausalVQA* (arXiv 2025)](https://arxiv.org/abs/2506.09943)  
    用途：`§45`、`§47`。负证据，说明反事实与预期问题仍显著困难。

## 六、图像中的世界状态：深度、几何、对应、布局

29. [Apple, *Depth Pro: Sharp Monocular Metric Depth in Less Than a Second* (official research page, October 2024)](https://machinelearning.apple.com/research/depth-pro)  
    用途：`§46`。把单图 depth/boundary 做成 foundation model 级问题。

30. [Apple, *Matrix3D: Large Photogrammetry Model All-in-One* (official research page, May 2025)](https://machinelearning.apple.com/research/large-photogrammetry-model)  
    用途：`§46`。把 pose/depth/view synthesis 压进统一模型。

31. [Google DeepMind, *D4RT: Teaching AI to see the world in four dimensions* (official post, 2026-01-22)](https://deepmind.google/blog/d4rt-teaching-ai-to-see-the-world-in-four-dimensions/)  
    用途：`§46`。说明 4D scene reconstruction / tracking 已成为 query-based 表征问题。

32. [Bordes et al., *V-JEPA 2.1* (arXiv 2026)](https://arxiv.org/abs/2603.14482)  
    用途：`§46`。把 dense features、depth、navigation、robot grasping 接回预测式表征框架。

## 七、终极残差：表征度量为什么会成为主问题

33. [Zhai et al., *A Large-scale Study of Representation Learning with the Visual Task Adaptation Benchmark* (VTAB, arXiv 2019)](https://arxiv.org/abs/1910.04867)  
    用途：`§47`。把“好表征”重新定义为对多样未见任务的适应能力。

34. [Locatello et al., *Challenging Common Assumptions in the Unsupervised Learning of Disentangled Representations* (arXiv 2018 / ICML 2019)](https://arxiv.org/abs/1811.12359)  
    用途：`§47`。说明解释变量式表征一旦被讨论，就会撞上归纳偏置与可识别性问题。

35. [Kornblith et al., *Similarity of Neural Network Representations Revisited* (arXiv 2019)](https://arxiv.org/abs/1905.00414)  
    用途：`§47`。CKA 系列工作的基础文本，说明“表征相似性”本身就是方法学问题。

36. [Ding et al., *Grounding Representation Similarity with Statistical Testing* (arXiv 2021)](https://arxiv.org/abs/2108.01661)  
    用途：`§47`。说明不同表征相似度度量会得出相反结论，度量不是透明窗口。

## 九、一句话结论

> 阶段 V 现在的主线已经可以被一套明确文献支持起来：先用重建桥接边界，再用不变性与自蒸馏立底座，再把 dense structure 压成对象与对应，随后用 JEPA 把学习对象切到状态预测，最后在 VTAB / disentanglement / representation similarity 处撞上度量本身。
