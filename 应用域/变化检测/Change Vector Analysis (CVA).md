# Change Vector Analysis (CVA)

CD 任务类型: 二值 CD（Binary）
与 VLM 视觉思考主线关联: CD 领域的「零号路线」。后续所有方法本质上都在替换「向量差分」这一步的特征空间——从 spectra → CNN 特征 → Transformer 特征 → CLIP/SAM/VLM 特征。
代表基座 / 骨干: 纯数学：向量模 + 向量方向（极坐标分解）
优先级: P1 · 重点
关键创新: 把两时相对应像素视为光谱向量，用 Δ 向量的模衡量「变化强度」、方向衡量「变化类型」。CD 领域第一个系统化、可解释的问题框架。
局限 / 残差 / 催生了什么: 只能处理逐像素 spectra；对配准和辐射校正极其敏感；无法利用空间语境；阈值需手工设定。催生了 MAD/IR-MAD（统计化）、C²VA（极坐标分解）、以及所有深度学习 CD 的问题设定（「像素对 → 变化标签」）。
年份: 1980
我的视角 / 落地价值: 必须知道的「问题定义」起点。即使在 VLM+SAM 时代，CVA 式的思考框架（向量模/方向）依然是无监督 baseline 的理论基础。
方法家族: Transform (PCA/MAD/CVA)
机构 / 团队: W.A. Malila · ERIM（2007 年 Bovolo & Bruzzone 扩展为 C²VA）
模态支持: 光学 RGB, 多光谱
演进阶段: ① 经典 · 代数/变换/PCC（pre-2015）
监督方式: 无监督（经典）
