# 四、第二条必须补：DINOv3 不是“又一个 DINO”，而是 dense feature scaling 的拐点

这件事在你的大厦中意义很大：

§43-B DINOv3：dense feature scaling 的正面化

# 六、第四条必须补：几何 foundation model 应成为 §46 的主证据，不只是例子

你现在 §46 讨论图像中的世界状态，但需要更多 2024–2026 的几何前沿支撑。这里建议拆成三层：

```text
§46-A 单图几何状态：Depth Pro / Depth Anything V2 / Metric3D / UniDepth
§46-B 多视图几何状态：DUSt3R / MASt3R / VGGT / Matrix3D
§46-C 持续 3D/4D 状态：CUT3R / D4RT / V-JEPA 2.1
```

## 6.1 单图几何状态

Depth Pro 是非常好的证据，因为它把单图 metric depth 做成 foundation model 问题。Apple 官方页面称 Depth Pro 是 zero-shot metric monocular depth foundation model，能输出带绝对尺度的高分辨率 depth map，不依赖 camera intrinsics，并且能在标准 GPU 上 0.3 秒生成 2.25MP depth map；其技术贡献包括 multi-scale ViT、真实+合成数据训练协议和 dedicated boundary metrics。([Apple Machine Learning Research][5])

这应该落到 §46：

```text
Depth Pro 说明：
“几何深度”已经不是传统下游任务，而是图像世界状态的一类 foundation object。
```

Depth Anything V2、Metric3D v2、UniDepth 也应该进入同一小节：Depth Anything V2 强调 synthetic data、teacher scaling 和 pseudo-labeled real images；Metric3D v2 把 zero-shot metric depth 与 surface normal 合为几何 foundation model；UniDepth 则直接预测 metric 3D points，并强调跨域 metric depth。([arXiv][6])

建议写成：

```text
单图几何状态不再只是 depth map。
它包含 metric scale、camera ambiguity、boundary sharpness、surface normal、focal/camera inference。
```

## 6.2 多视图几何状态

DUSt3R、MASt3R、VGGT、Matrix3D 是 §46 必须补的重头。

DUSt3R 把任意图像集合的 3D 重建改写成 pointmap 回归，即不依赖相机内参和位姿先验地从图像预测稠密 3D 表达；MASt3R 在 DUSt3R 基础上加 dense local features 和 matching loss，把匹配重新接回 3D；VGGT 则进一步提出 feed-forward 网络，直接从一张、少量或数百张视图推断 camera parameters、point maps、depth maps 和 3D point tracks，并在多项 3D 任务上达到 state-of-the-art。([arXiv][7])

Matrix3D 也很关键。Apple 页面称 Matrix3D 用一个统一模型处理 pose estimation、depth prediction、novel view synthesis，并用 multi-modal DiT 融合 images、camera parameters、depth maps 等模态；其 mask learning strategy 允许用不完整模态数据训练。([Apple Machine Learning Research][8])

这部分建议新增：

```text
§46-B 多视图几何状态：从 depth map 到 pointmap / pose / track
```

核心命题：

> **阶段 V 的世界状态不是一句“有 geometry”，而是图像系统开始内部持有 camera、depth、pointmap、pose、track、view consistency 这些状态变量。**

## 6.3 持续 3D/4D 状态

CUT3R 和 D4RT 是“状态持有”非常直观的证据。

CUT3R 摘要明确说它是一个能连续更新内部 state representation 的 recurrent framework，可以从图像流在线生成 metric-scale pointmaps，并把 pointmaps 累积到共同坐标系中；它还能通过 virtual unobserved views 推断不可见区域。([arXiv][9])

D4RT 是 Google DeepMind 2026 年 1 月发布的统一 4D scene reconstruction and tracking 模型，官方介绍强调它从视频这种 2D 投影中恢复/理解运动中的 3D 世界，并使用 query-based 机制回答“某个视频像素在任意时间、任意相机视角下位于 3D 空间何处”的问题；DeepMind 还称它比以往方法最高可高效约 300 倍。([Google DeepMind][10])

这应该成为 §46 的最前沿小节：

```text
§46-C 持续状态：从图像状态到 4D 世界片段
```

但要加边界：

```text
这已经越过单图图像状态，进入视频/4D 外推接口。
在阶段 V 主体中，它作为“状态持有向时空世界模型扩张”的证据，而不是本章核心定义的替代。
```

# 七、第五条必须补：聚合表征模型是阶段 V 的“多教师闭合制度”

你当前文献里有 AM-RADIO / RADIOv2.5 / C-RADIOv4，但它们没有在主线中获得足够理论位置。它们非常适合补一个新概念：

> **聚合表征闭合：当单一训练目标不再足以覆盖所有状态，系统开始蒸馏多个教师的不同能力。**

C-RADIOv4 技术报告说，它继承 AM-RADIO 的 agglomerative foundation model 思路，通过 multi-teacher distillation 把多个异构模型的能力合成一个学生模型；C-RADIOv4 的教师包括 SigLIP2、DINOv3 和 SAM3，并进一步改善 any-resolution support、高分辨率效率、dense perception 和 text alignment。([arXiv][11])

Perception Encoder 也非常重要。它提出 PE 家族，并指出强视觉 embedding 不一定在网络输出层，而隐藏在中间层；它通过 language alignment 和 spatial alignment 把这些 embedding 拉出来，服务多模态语言建模和 dense prediction，并报告图像/视频分类检索、DocVQA/InfographicVQA/PerceptionTest，以及 detection、tracking、depth 等 spatial tasks 的强结果。([arXiv][12])

SigLIP2 则表明图文编码器路线本身也在向 localization 和 dense prediction 靠拢；它在原始 image-text 目标之外加入 captioning-based pretraining、自蒸馏、masked prediction 和 online data curation，并报告 localization / dense prediction 改进。([arXiv][13])

建议新增：

```text
§43-C 聚合表征：从单一目标到多教师闭合
```

核心判断：

```text
DINOv3：纯视觉 dense self-supervision。
SigLIP2：语言-视觉编码器向 dense/localization 回补。
RADIO/C-RADIO：把 DINO/SigLIP/SAM 等能力蒸馏进一个 encoder。
PE：发现最好 visual embeddings 可能在中间层，需要 alignment 才能被不同接口调用。
```

理论意义：

> **阶段 V 的前沿不是“哪个 encoder 赢”，而是多个闭合制度开始被压缩进同一个视觉表征。**

这会把阶段 V 和阶段 III/II 的桥接说清。

# 八、第六条必须补：负证据与反身评测

阶段 V 不能只列强模型，否则会变成“乐观世界模型史”。你需要专门写“负证据”：

```text
§47-A 状态持有的负证据：为什么 frontier 模型仍不懂物理/因果
```

IntPhys2 是关键负证据。它以 permanence、immutability、spatio-temporal continuity、solidity 等核心原则测试视频模型的 intuitive physics，并报告当前模型在复杂场景的直觉物理理解上仍然明显落后于人类，许多模型接近 chance level。([arXiv][14])

CausalVQA 也关键。它用 real-world scenarios 的问题来测试视频模型的 causal reasoning，包括 counterfactual、hypothetical、anticipation、planning 和 descriptive 等问题类型，并指出当前 frontier multimodal models 在 anticipation 和 hypothetical 问题上显著低于人类。([arXiv][15])

这说明：

```text
深度、pointmap、track、dense feature 强，不等于物理/因果状态闭合。
```

建议 §47 加一张表：

| 负证据                              | 测什么                                       | 暴露什么                 |
| -------------------------------- | ----------------------------------------- | -------------------- |
| IntPhys2                         | object permanence / solidity / continuity | 视觉状态不等于直觉物理          |
| CausalVQA                        | counterfactual / anticipation / planning  | 状态持有不等于因果推理          |
| probe 排序反转                       | linear / dense / geometry / physics       | 好表征不可由单一指标裁判         |
| DINO dense artifacts / registers | dense feature map artifact                | dense feature 自身需要审计 |
| V-JEPA 2.1 dense gains           | 预测式 dense state 有进展                       | 但还需多状态对象验证           |

这里的理论结论：

> **阶段 V 的终极残差不是“模型还不够强”，而是“我们还不知道用什么制度证明它真的持有世界状态”。**

# 最核心的一句话

阶段 V 现在要补的不是更多“自监督论文”，而是：

> **把 2024–2026 的前沿全部改写成“状态对象被谁持有、持有到哪一级、用什么证据裁判、留下什么残差”。**

DINOv3 证明 dense feature scaling 成为中心问题；V-JEPA 2.1 把预测式表征推向 dense 和时空 grounding；Depth Pro、Matrix3D、VGGT、CUT3R、D4RT 把几何、pointmap、pose、track、4D scene 推成 foundation-level 状态对象；IntPhys2 和 CausalVQA 则提醒我们：几何强、dense 强、规划接口强，仍不等于物理/因果世界状态真正闭合。([arXiv][3])

所以阶段 V 的新标题感可以是：

> **表征共形：视觉开始把可见性保存为状态对象，并在几何、预测、4D 与度量危机中逼近真正的世界持有。**
