# 阶段偏序与桥接节点

类型: 协议/阶段结构
阶段: 横切
关键贡献: 把五阶段从“时间线”重写为“主导闭合型态的偏序结构”，并给跨阶段方法、桥接节点和反向输血提供统一判定规则。
残差 / 它催生了什么: 解决技术史线性化风险；催生主阶段/副阶段/桥接节点/反向输血四个定位标签。
定位标签: 阶段偏序, 桥接节点, 主阶段, 副阶段, 反向输血, 闭合型态

五阶段不是历史年表。

它们是五种主导闭合型态：

| 阶段 | 主导闭合型态 | 最小问题 |
|---|---|---|
| I · 判别共形 | 标签接口闭合 | 整图差异如何压成可训练标签 |
| II · 稠密共形 | 空间场闭合 | 表征如何在图像定义域内处处做功 |
| III · 语言共形 | 公共语义闭合 | 视觉如何进入可共享、可调用的语义空间 |
| IV · 生成共形 | 反向生成闭合 | 内部状态如何反向产出观测 |
| V · 表征共形 | 世界状态与度量反身闭合 | 系统如何持有状态并审计度量 |

它们之间存在残差驱动的偏序关系，但不是严格时间顺序。一个方法可以同时触碰多层楼；它的主阶段取决于它主要扩张了哪一种闭合。

---

## 一、四个定位标签

### 1. 主阶段

主阶段指一个方法主要解决的闭合残差。

判定问题：

```text
这个方法最主要扩张了哪一种闭合半径？
它吃掉的是哪一层的出口残差？
```

### 2. 副阶段

副阶段指一个方法借用、调用或局部增强的其他闭合能力。

判定问题：

```text
它借用了哪一层已经形成的闭合？
它是否只是把另一层能力接进来，而没有改变该层主命题？
```

### 3. 桥接节点

桥接节点指无法稳定归入单层、但改变了层间关系的方法或机制。

判定问题：

```text
如果删除这个方法，两个阶段之间的过渡是否会变得难以解释？
它是否把前一阶段的残差转译为后一阶段的入口对象？
```

### 4. 反向输血

反向输血指高阶段机制回流增强低阶段任务。

判定问题：

```text
高阶段的表征、生成、语言或度量能力是否回流改善了低阶段的检测、分割、分类或检索？
这个回流是否改变低阶段任务的闭合方式？
```

---

## 二、典型桥接与跨层关系

| 方法/路线 | 主阶段 | 副阶段 | 结构判断 |
|---|---|---|---|
| SAM | II | III | 稠密场闭合 + prompt 接口闭合；不等于语言语义完全闭合 |
| Grounding DINO / OWL-ViT | III | II | 语言语义进入检测接口，语言闭合回压稠密任务 |
| CLIP dense adaptation | III | II | 公共语义闭合增强稠密场，但语义仍借语言空间 |
| MAE / BEiT | IV -> V 桥 | I/II | 重建式生成目标反身成表征，为非生成表征转向铺路 |
| DINO / iBOT | V | II | 表征持有回流到 dense structure，属于 V 对 II 的反向输血 |
| diffusion features | IV -> V 桥 | II/V | 生成过程中的中间激活回流做表征与对应 |
| VLM | III | II/V | 语言闭合强，但视觉空间闭合和世界状态持有可能弱 |
| image editing | IV | III/V | 生成闭合中加入语言意图和不变量保持，逼近状态持有问题 |

## 三、阶段间不是单向取代

阶段后发不意味着前一阶段失效。

正确读法：

```text
I 不是被 II 淘汰，而是被 II 要求空间化。
II 不是被 III 淘汰，而是被 III 要求语义开放。
III 不是被 IV 淘汰，而是被 IV 要求反向生产。
IV 不是被 V 淘汰，而是被 V 要求从产出转向持有。
```

因此，后续写作中不应使用“进入下一阶段所以上一阶段过时”的叙事。应写成：

> 上一闭合型态被后一闭合型态重新约束。

## 四、定位决策表

| 问题 | 如果答案是 | 定位 |
|---|---|---|
| 主要解决的是整图标签或训练动力学吗？ | 是 | I |
| 主要解决的是像素、框、mask、dense query 吗？ | 是 | II |
| 主要让视觉进入语言或公共语义接口吗？ | 是 | III |
| 主要从内部产出、重建、编辑观测吗？ | 是 | IV |
| 主要学习非生成表征、状态预测、对象/几何/对应持有吗？ | 是 | V |
| 同时触碰两层且改变过渡关系吗？ | 是 | 桥接节点 |
| 用高阶段能力增强低阶段任务吗？ | 是 | 反向输血 |

## 五、写作协议

新增论文、模型或应用域案例时，至少给出：

```text
主阶段:
副阶段:
桥接/反向输血:
主导闭合型态:
吃掉的残差:
留下的残差:
```

如果一篇材料只能按年份归类，却不能按闭合型态归类，它还没有真正进入这座大厦。
