# 1. 阶段 IV 当前最大问题：公式没有“契约”

你现在已经意识到自己对公式、参数、系数不敏感，这很关键。阶段 IV 里公式确实比前三阶段更密集，因为生成模型本来就是概率、潜变量、动力学、采样过程交织的一层。但公式本身不是问题，问题是：**公式出现时，还没有稳定回答读者最需要的五件事。**

# 4. 阶段 IV 的公式应全部改成“最小必要公式”

你现在几个公式有明显渲染或书写问题，这会加重你的阅读负担。

## 4.1 GAN 公式需要修

§32 当前 GAN 原始目标公式渲染为：

```text
E{x ~ p[\log D(x)] ...
```

这是坏的。应改成：

```latex
\min_G \max_D V(D,G)
=
\mathbb{E}_{x \sim p_{\text{data}}}
[\log D(x)]
+
\mathbb{E}_{z \sim p_z}
[\log(1-D(G(z)))].
```

非饱和生成器损失可以写：

```latex
\min_G
-
\mathbb{E}_{z \sim p_z}
[\log D(G(z))].
```

这样 §32 的意思才清楚：GAN 是一个同时训练生成器 `G` 和判别器 `D` 的二人 minimax 游戏，原始论文也把它定义为 adversarial process / two-player game。([CallMeCaicai][4])

但你不要让读者死磕公式。公式卡应该写：

```text
一句话：
G 负责造图，D 负责挑错。G 的方向来自 D 的挑错。

关键旋钮：
D 太强，G 没梯度；
D 太弱，G 学不到边界；
二者要处在“互相逼迫但不互相杀死”的区间。
```

这比单纯写 minimax 更有用。

---

## 4.2 VAE / β-VAE 公式需要修

§33 当前 ELBO 写成：

```text
\mathcal{L}{ELBO}
```

β-VAE 写成：

```text
\mathbb{E}{q\phi(z \mid x)}
```

这些公式会让不熟悉数学的读者更痛苦。建议修成：

```latex
\mathcal{L}_{\text{ELBO}}
=
\mathbb{E}_{q_\phi(z|x)}
[\log p_\theta(x|z)]
-
D_{\text{KL}}
(q_\phi(z|x)\,\|\,p(z)).
```

β-VAE：

```latex
\mathcal{L}_{\beta\text{-VAE}}
=
\mathbb{E}_{q_\phi(z|x)}
[\log p_\theta(x|z)]
-
\beta
D_{\text{KL}}
(q_\phi(z|x)\,\|\,p(z)).
```

VAE 原论文的关键贡献之一就是用 reparameterization trick 让变分下界可以用标准随机梯度优化，这对应你 §33 中“把随机潜空间工程化”的判断。([CallMeCaicai][5])

公式卡应该写：

```text
一句话：
VAE 一边要求 z 能重建图像，一边要求 z 不要乱长，要贴近一个简单先验。

第一项：
z 要能解释 x。

第二项：
z 要规整，方便采样。

β：
β 越大，z 越规整，但重建越容易损失细节。
β 越小，重建越好，但 latent 更混乱。
```

这样读者就不需要先理解完整变分推导，也能懂 VAE 为什么属于“内部语闭合”。

## 4.3 DDPM 公式需要修，并分层讲

§34 当前 `\mathcal{L}{simple}` 需要修成：

```latex
\mathcal{L}_{\text{simple}}
=
\mathbb{E}_{t,x_0,\epsilon}
\left[
\|\epsilon -
\epsilon_\theta(x_t,t)\|^2
\right].
```

DDPM 的核心确实是前向逐步加噪、反向逐步去噪，并常用噪声预测参数化；Ho 等人的论文也把 diffusion probabilistic models 写成受非平衡热力学启发的 latent variable model，并使用噪声预测式训练。([CallMeCaicai][6])

但你应该把 §34 的公式拆成三层：

### 第一层：不用公式

```text
把清晰图像逐步加噪；
训练模型在每一步猜出噪声；
采样时从噪声一步步去掉噪声，图像逐渐显形。
```

### 第二层：五个变量

```text
x_0：原图
x_t：第 t 步的带噪图
t：噪声等级 / 内部时间
ε：真实加入的噪声
ε_θ：模型预测的噪声
```

### 第三层：一个公式

```latex
\|\epsilon - \epsilon_\theta(x_t,t)\|^2
```

然后写：

```text
这个公式只是说：
模型猜的噪声，和真实噪声越接近越好。
```

这会让 §34 的门槛低很多。

# 5. §34 是阶段 IV 主轴，但 SDE / Flow Matching 应该标为“第二遍阅读”

§34 是你阶段 IV 最重要的一章。你现在说得很对：扩散不是“更稳的 GAN”，而是把生成从一次映射改成生成动力学时间；并且你已经限定这不是世界事件时间，而是样本显形的算法时间。这个边界非常重要。([CallMeCaicai][6])

你可以在 §34 开头加一个提示：

## 11.1 修复 REPA 公式

当前 REPA 公式写成：

```latex
\mathcal{L} = \mathcal{L}{diffusion} + \lambda \mathcal{L}
```

它缺少下标。建议改成：

```latex
\mathcal{L}
=
\mathcal{L}_{\text{diffusion}}
+
\lambda
\mathcal{L}_{\text{align}}.
```

然后解释：

```text
第一项：仍然学会生成。
第二项：让生成器中间表征靠近强视觉表征器。
λ：表征对齐的权重。
λ 太大，生成目标可能被表征目标压住；
λ 太小，表征校准不起作用。
```

## 11.2 G1/G2/G3 表格要正式渲染

§37 定位卡已经区分 G1 feature usefulness、G2 representation transfer、G3 state holding，这个非常好，但目前未渲染成表格。([CallMeCaicai][14])

建议正式写成：

| 层级 | 名称                      | 含义                     | 阶段地位        |
| -- | ----------------------- | ---------------------- | ----------- |
| G1 | feature usefulness      | 生成器中间特征对分割、对应、编辑等任务有用  | IV 内部证据     |
| G2 | representation transfer | 特征能迁移到多个视觉接口           | IV → V 桥接   |
| G3 | state holding           | 特征稳定持有对象、几何、遮挡、对应等状态变量 | 才进入 V 的主体命题 |

这张表必须保留。它能防止你把“扩散特征好用”直接误写成“世界状态已持有”。

# 15. 阶段 IV 最应该新增的两张总表

## 15.1 “生成家族对照表”

| 家族             | 学什么              | 生成怎么发生                    | 强项            | 弱项            |
| -------------- | ---------------- | ------------------------- | ------------- | ------------- |
| EBM            | 能量地形             | 采样落入低能区                   | 概念清楚          | Z 和采样难        |
| Autoregressive | 条件概率链            | 一个 token/像素 接一个           | likelihood 清楚 | 慢，顺序僵硬        |
| GAN            | 判别边界             | G 与 D 对抗                  | 锐利、感知强        | 不稳、模式崩        |
| VAE            | 潜变量              | z 解码成图                    | 可采样、内部坐标      | 发糊、控制弱        |
| VQ-VAE         | 离散码本             | token/code 解码             | 离散内部语         | codebook 语义不稳 |
| Flow           | 可逆变换             | 噪声可逆映到图像                  | likelihood 严格 | 表达成本高         |
| Diffusion      | 去噪轨迹             | 从噪声逐步显形                   | 稳定、高质、可控      | 慢、工艺复杂        |
| LDM            | latent diffusion | latent 里扩散                | 成本低、产品化       | VAE 压缩残差      |
| DiT/Flow       | token/速度场        | transformer/velocity 组织轨迹 | scaling 潜力    | 工艺仍发展         |

## 15.2 “公式旋钮表”

| 符号   | 在哪里               | 它是什么           | 直觉          |
| ---- | ----------------- | -------------- | ----------- |
| `z`  | GAN/VAE/LDM       | 内部随机种子/潜变量     | 从哪里开始生成     |
| `β`  | VAE/DDPM          | KL 权重或噪声步长     | 规整/噪声强度     |
| `Z`  | EBM               | 全空间归一化         | 难算的大锅       |
| `D`  | GAN               | 判别器            | 外部抵抗        |
| `KL` | VAE               | 分布差异           | latent 是否规整 |
| `ε`  | DDPM              | 噪声             | 要猜出的杂质      |
| `t`  | Diffusion         | 生成时间/噪声等级      | 现在图像有多脏     |
| `s`  | CFG               | guidance scale | prompt 推力   |
| `λ`  | REPA/Control/Loss | 附加约束权重         | 约束有多强       |
