# 文献查验工具

用于核验前三阶段 Markdown 中出现的论文引用，目标是尽量把以下字段落成可审计台账：

- 标题
- 作者
- 年份
- 发表 venue / 期刊 / 会议
- 出版机构 / 承载源
- 官方落地页

当前脚本：

- `scripts/audit_stage123_references.py`

当前输出：

- `横切/文献查验/前三阶段文献查验总表.csv`
- `横切/文献查验/前三阶段文献查验摘要.md`
- `横切/文献查验/前三阶段arXiv核验.md`

设计原则：

- `arXiv` 号一律先走 arXiv 官方 API 核验。
- 标题化引用优先用 OpenAlex 做题名匹配与 metadata 汇总。
- 输出里会显式区分 `已核验`、`仅 arXiv 核验`、`待人工复核`、`字段不一致`。
