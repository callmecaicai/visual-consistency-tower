# Change Matrix

## 1. New paper strengthens an existing chapter

Write target:

- existing `02+` file, or create a new `02-扩展稿-*`

Avoid:

- editing root navigation first
- creating a new chapter

## 2. New paper is domain-specific evidence

Write target:

- `应用域/<domain>/`

Also record:

- main stage
- main chapter anchor

Avoid:

- forcing domain evidence into a canonical chapter body

## 3. New paper changes the chapter's core mechanism

Write target:

- `00-定位卡.md`
- `01-机制展开.md`

Also consider:

- update stage navigation

## 4. New theme has no valid host chapter

Write target:

- new `§xx-*` directory, but only after checking stage bandwidth and residual logic

Also require:

- stage-level justification
- navigation update

## 5. Incoming file is an export, draft, or old fragment

Write target:

- `归档`

Avoid:

- mixing into canonical directories

## 6. Task only needs agent guidance, not repository edits

Write target:

- no repository edit
- emit positioning card or update note only
