# 派生判据与边界

类型: 公理层 / 派生判据
阶段: 00 · 公理层
共形维度: 表征, 语义, 度量, 主体性, 反身
关键贡献: 把表征、理解、半径、因果、符号、语义链等派生命题压成一份判据清单，避免公理层膨胀为第二套正文。
残差 / 它催生了什么: 本文件只保留能直接服务五问协议和阶段定位的硬判据；历史叙事、重复解释和长篇推演不再放在公理层。
定位标签: 派生判据, 表征闭合, 理解, 世界模型, 因果, 符号, 一致性半径, 语义链

本文件不是新公理。

它只回答一个问题：

> **从“真正的开端”和“第一性原理”出发，哪些派生命题仍然值得保留为全站判据？**

若一个命题不能帮助判断章节的纯存在态、稳定差异、闭合制度、成功遮蔽和内在残差，它就不应继续留在公理层。

---

## 一、最小认知链

视觉、语义、语言、功能、智能不是五个并列模块，而是一条从差异到重写的链。

| 环节 | 极简定义 | 判定问题 |
|---|---|---|
| 视觉 | 可见性中的差异被系统捕获 | 系统到底保留了什么差异？ |
| 表征 | 稳定差异进入内部结构 | 差异保存在哪里？ |
| 语义 | 表征获得跨域闭合地位 | 它能否跨任务、跨接口、跨变换被调用？ |
| 语言 | 语义进入公共编码与调用制度 | 它是公共载体，还是语义本体？ |
| 功能 | 语义在某个接口上的局部闭合 | 成功是否只限于单一接口？ |
| 智能 | 系统在内在残差压力下重写链条 | 失败能否反过来改写表征、接口、度量或动力？ |

最低纪律：

> **会说不等于有语义；会做不等于有智能；能闭合一个接口不等于持有世界。**

---

## 二、表征闭合判据

表征不是向量本身，而是被系统保留下来的稳定差异。

表征进入语义地位，至少要经过三层闭合：

| 层级 | 名称 | 判据 |
|---|---|---|
| B1 | 保存闭合 | 差异能在内部结构中被保留 |
| B2 | 调用闭合 | 差异能通过至少一个接口被稳定调用 |
| B3 | 跨域闭合 | 差异能跨任务、模态、尺度或时间维持相容 |

经验闭环不是“数据更多”，而是：

```text
数据产生差异
  ↓
表征保存差异
  ↓
接口调用差异
  ↓
度量暴露残差
  ↓
残差改写数据、表征、接口或度量
```

因此，大模型时代真正变化的不是“模型变大”本身，而是数据、参数、算力、监督、反馈与部署后残差开始进入同一个循环。

## 三、一致性半径判据

一致性不是输出相同，而是内部世界不裂。

一致性半径不是单一分数，而是系统在一组变换族、接口族和度量族上维持闭合的结构化能力。

```text
R(S) = {
  R_view,
  R_scale,
  R_noise,
  R_context,
  R_modality,
  R_semantic,
  R_temporal,
  R_edit,
  R_generation,
  R_action,
  R_metric
}
```

这些半径不等价。`R_view` 处理视角，`R_semantic` 处理开放概念，`R_temporal` 处理延续，`R_action` 处理感知、语言、计划和行动是否属于同一个世界，`R_metric` 处理度量被审计后系统是否仍能保持理论一致。

使用纪律：

> **闭合半径扩张，不等于某个 leaderboard 分数提高；它要求同一内部表征能承受更大的差异、接口和残差压力。**

## 四、理解与世界状态判据

理解不是答对题，也不是输出漂亮文本或样本。

最低定义：

> **理解 = 内部表征能在相关变换下保持相容，并能支持预测、解释、生成、编辑或行动中的多接口闭合。**

世界模型也不应被泛化成任何好用表征。

| 概念 | 最低判据 | 不足之处 |
|---|---|---|
| 好特征 | 下游任务有效 | 可能只是接口技巧 |
| 语义表征 | 跨域可调用 | 可能仍借语言或标签制度 |
| 隐式世界模型 | 能稳定预测未见状态或目标表征 | 不一定有因果、反事实或行动后果 |
| 图像世界状态 | 持有受证据约束的对象、几何、遮挡、对应等状态 | 单图状态天然欠定 |
| 反身世界模型 | 能审计哪些度量命中了状态，哪些度量误判状态 | 阶段 V 尚未闭合 |

阶段 V 的关键不是“再做更强 feature”，而是回答：

> **这个内部状态到底持有什么？通过什么接口被调用？在哪些变换下仍成立？被什么度量误判？**

## 五、因果与符号边界

本大厦不把因果还原为相关性。

当前只主张：

1. 视觉系统首先学习的是稳定相容结构。
2. 因果结构需要更强的干预、反事实、时序和行动约束。
3. 不变性可以提示因果候选，但不自动等于因果。
4. 因果不是阶段 I-V 主线的已完成对象，而是阶段 V 与反身之后的高阶残差。

符号也不是裸 token。

符号至少需要三重约束：

| 约束 | 含义 |
|---|---|
| 接地 | 能回到视觉、行动、任务或共同体实践 |
| 角色 | 在推理、组合、调用中有稳定功能 |
| 规范 | 有被纠错、共享、审计的公共制度 |

因此：

> **语言 token 可以承载符号，但 token 本身不是符号；符号是被接地、被使用、被规范化的关系结构。**

## 六、可证伪判据

这些判据必须能进入预测登记和验证协议。

| 编号 | 命题 | 可证伪方式 |
|---|---|---|
| A1 | 高任务分数不等于跨域闭合 | 多接口评估出现排序反转 |
| A2 | 语言对齐会压扁未被语言高频编码的视觉差异 | 去语言或弱语言条件下细粒度视觉显著退化 |
| A3 | 一致性半径不是单榜分数 | 多半径 dashboard 中模型优势不可互相替代 |
| A4 | 好表征不等于世界状态持有 | probe 强但 depth / correspondence / occlusion 弱 |
| A5 | 因果需要干预与反事实约束 | 纯相关模型在分布外反事实任务中系统失败 |
| A6 | 功能闭合不等于智能 | 高成功率系统在接口失配时不能重写策略 |
