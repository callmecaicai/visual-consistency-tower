## 投影表面哲学

阶段 II 的哲学问题不是“分割意味着什么”，而是：

> 投影表面如何被误认为世界？

FCN、DeepLab、DETR、MaskFormer、SAM 都在 `Ω` 上建立空间法律。它们能让每个像素、区域、对象和 prompt 得到回答，却仍只是在传感器平面上闭合。哲学段落的作用，是不断提醒：投影不是本体，空间输出不是公共意义，表面可度量也不是世界状态持有。

### 1. 柏拉图的洞穴：视觉模型只见投影

柏拉图说，洞穴里的人背对火光，只能看见物体在壁上的**投影**，误以为那就是实在。

阶段二的视觉模型，恰恰是这样的囚徒——它们永远看到的只是 **3D 世界在二维 sensor plane 上的投影**。不论 SAM 的 ViT-H、DINOv3 的 ViT-7B、还是 Florence-2 的 seq2seq，输入都是 $H\times W\times 3$ **的像素网格**——一个已经丢失了深度、光场、观察者视角可选性的**平面投影**。

- COCO / SA-1B / LVD 中的每一张图，都是**某个观察者**在**某个瞬间**的**某个角度**下拍下的一次记录
- 模型看不到这张图**的另一面**，看不到拍下它的人**为什么拍**，看不到拍摄前一秒发生了什么
- 但模型被期待从这个投影中，恢复出整个本体结构：物体身份、几何、意图、因果

这是一个**认识论上的结构性不匹配**——任务要求模型回答的问题（What is this? Where is it? How does it work?）在本体层属于三维时空，而模型得到的只是度量后的二维片段。

**DINO 和 SAM 的所有胜利都是在洞形投影的层次上的胜利**——它们把投影之间的统计一致性提到了极致，但没有离开洞形。

### 2. 康德：先天综合判断的缺席

康德的《纯粹理性批判》问：我们为什么能认识一个我们从未经验过的事物？因为我们具备**先天的感性形式**（空间、时间）和**先天范畴**（因果、实体、统一性）。

视觉模型有什么先天结构？

| 先天结构 | 阶段一 | 阶段二 | 深层残差 |
| --- | --- | --- | --- |
| 时间 | 没有 | SAM 2 / V-JEPA 2 的帧间记忆 | 但**只能储存**，不能**先验地结构化**过去-现在-未来 |
| 实体 | 没有 | object query / instance query | 但**实体是被学出来的**，不是先验的 |

视觉模型没有真正的康德式**先天范畴**。它的「实体感」、「连续感」、「因果感」都是从数据中**估计**出来的统计规律，不是结构性的前提。

这解释了为什么 MMVP 的尴尬：让模型判断「鱼是实体还是纹理」、「镰子能支撑 300 斤吗」，它没有先验的**实体论**、**力学**、**因果论**来支撑，只能查统计共现。

### 3. 维特根斯坦：语言游戏的借用与归属

后期维特根斯坦说：「词的意义在其**用法**中」——语言游戏（language game）的参与才赋予概念以意义。

观察阶段二的开集识别：

- Grounding DINO 的「检测这张图里的 person」——`person` 的意义**不是视觉模型学的**，而是 CLIP 从 400M 对「人+图」杂乱的文本对齐里学的
- CLIP 的 `person` 语义，其**游戏**是文本分布——人类怎么用这个词写图说明、怎么做alt-text、怎么发照片配文
- 即 `person` 的意义**属于人类语言游戏**，视觉模型只是**进入这个游戏的外围**

维特根斯坦的判断：视觉模型**没有自己的语言游戏**。它似乎有、实际上是人类语言游戏的影子。

> 视觉模型说不出「这个物体」——它只能说 $\text{similarity}(\text{视觉嵌入}, \text{化为 `cat` 的文本嵌入}) = 0.87$。尽管结果相似，但**意义的住所在文本侧**，不在视觉侧。
> 

阶段三要问的是：视觉能不能有**自己的语言游戏**？JEPA 的尝试就是把「游戏规则」换成「在嵌入空间预测」——但这个游戏是否能产生真正的「视觉概念」，仍待开放。

### 4. 梅洛-庞蒂：身体缺席的知觉

梅洛-庞蒂在《知觉现象学》里强调：**知觉不是脑的计算，是身体在世界中的展开**。人看到的桶子，包含着「我可以伸手去握的无限可能」。

阶段二的视觉模型没有身体：

- SAM 看到杯子，输出 mask，但模型永远不知道**杯子距离它多远**、**杯子能不能被握**、**把杯子碰倒会怎样**
- Depth Anything 能估计深度，但**深度仅是一个标量字段**，不是身体可用性（affordance）
- Florence-2 能写「一个浅蓝色陶瓷杯」，但它没有**用杯子喝水的经验**来支撑这个 caption

梅洛-庞蒂的预言：**无身体的知觉**会收敛到「表面相似的干燥表层」——也就是现在 VLM 上见到的「流畅底下的平庸」。

这对应了阶段三的一个**独立趋势**：**Embodied AI**（VLA、RT-2、OpenVLA）——把视觉模型接到机器人身体上，让它从**投影**返回**世界**。

### 5. 海德格尔：在手 vs 上手的视觉

海德格尔区分**上手**（zuhanden，透明可用的工具）与**在手**（vorhanden，被凝视的客体）。

匹配到视觉模型：

- **上手视觉**：开车时我们不意识到正在观察路面——视觉直接融入驾驶行为，停下来说「这里有一个交通锥」才进入认知。人类的视觉绝大多数时候是上手的
- **在手视觉**：把图像放在显示器上，划框写标注，问「这是什么」——视觉被主题化、冷冻化、客体化

阶段二的所有模型，训练和评估都在**在手模式**。COCO 图片是被筛出的、平台上传的、人类觉得值得拍的。模型没有见过**上手的视觉流**——一个生物为了继续存活而持续接收的、流动的、绝大多数无用的视觉输入。

这解释了为什么阶段二结束时的模型，**放到机器人身上就坏掉**——它们不会处理**模糊、部分遮挡、视觉无人值守的普通瞬间**，因为它们从未在这种**生活世界**中生长。

### 6. 谢林：从「被认识的自然」到「能生成的自然」

这是最关键的哲学层趋势。阶段一结语里提到过谢林的「原初决断」——在谢林那里，自然并不是消极的、待人认识的对象，而是积极的、自我生成的主体。

阶段二的视觉模型走的是**康德路径**：把图像作为**待认识的现象**，用范畴（类别、框、mask）去裁剪它。整个范式是**认识论的**：模型是认识主体，图像是认识客体。

谢林提示另一条路——**生成的路**：

- 模型不是**识别**图像，而是**生成**图像
- 理解一个东西的标志不是能把它分类，而是能把它**画出来**、把它的另一个视角想象出来、把它缺席的部分补全
- **Sora / Stable Diffusion / Genie / Cosmos** 这条线是在此处**绕过**了康德路径

这意味着：视觉基础模型的真正终极形态，可能不是更强的 DINO 系列、更大的 SAM 系列，而是**世界模型**（World Model）——可以被提示生成、翻转、续写、想象。判别只是生成的一个特例（在所有可能的图像中，哪个最似？）。

> 谢林的调子：视觉应当从**反射性的认识**移向**建构性的生成**。
> 

### 7. 老子与禅：无名之见

《道德经》首章：「无名天地之始，有名万物之母。」

CLIP 的本体论位置是「有名」：万物的视觉中有名的（`cat`, `dog`, `bird`）才能被认识。未命名的视觉（漠然的纹理、人脸自然光影的微调、叶片表面的细节）在 CLIP 的视野里或多或少失离。

而视觉预训练的**自监督**路径（MAE / DINO / JEPA）恰恰在尝试回到「无名」：

- MAE 说：把图像遮掉 75%，剩下的能否预测缺席？这里没有名字
- DINO 说：同一张图的两个裁剪，能否达到相似的嵌入？这里没有名字
- JEPA 说：图的一部分能不能预测另一部分的嵌入？这里也没有名字

这一条线在阶段二中从未真正占上风——因为「有名」的 CLIP 路benchmark 更好看。但**MMVP 的出现暴露**：「有名」的视觉此刻已经到顶，继续 scaling 只能解决「更细粒度的有名」，不能解决**视觉本身的无名细节**。

禅宗用「不立文字」形容直接感知。如果视觉模型能有「不立文字」的能力——不通过语言知道一个东西——那才是真正的**视觉主体**。

### 8. 结：视觉作为主体的可能性

把上述七个试金石拼起来，阶段三的深层问题浮现：

<aside>
✴️

**视觉模型能不能成为一个认识主体？**

现在的答案是：不能。因为——

- 它只有投影（柏拉图），没有事物本身
- 它没有先验范畴（康德），只有统计共现
- 它使用人类的语言游戏（维特根斯坦），不带自己的
- 它没有身体（梅洛-庞蒂），存不起 affordance 结构
- 它只存在于在手模式（海德格尔），不知上手
- 它是被认识的自然（面对谢林），不是能生成的自然
- 它是「有名」的（对 CLIP，老子会追问其“无名”缺口），里面没有「无名的视觉」

阶段三的全部工作，其实是在这七个维度上给视觉模型「补身体」「补范畴」「补游戏」「补生成性」。

</aside>

### 一个更深的观察：视觉的共形论与存在论的双重同谋

阶段一在处理**共形论**（形式适配）。阶段二在扩张**共形论**（稠密适配）。但进入阶段三，**共形论已经不够**——因为所有能被共形的形式都被共形过了。

下一层问题属于**存在论**（ontology）——视觉模型在世界中**是什么**？它是：

- 一台函数？（只输入图输出 mask）
- 一个接口？（来什么 prompt 应什么）
- 一个世界模型？（模拟视觉世界的概率分布）
- 一个感知者？（主动收受世界的知觉主体）
- 一个 Agent？（在世界中行动、被反馈、修正信念）
- 或者是更深的某种东西？

阶段二的答案是前两项（函数 + 接口）。阶段三必须回答后面几项。

不解决这个存在论问题，视觉模型永远是**语言模型的眼睛**，而不是一种**独立的智能形态**。

## 九、最后一句

阶段一证明了**判别共形**（一张图 → 一个标签）是可行的；
阶段二证明了**稠密共形**（一张图 → 任意输出）是可行的；
而阶段三要回答的，是**视觉认知本身**——

> 视觉模型能不能**独立于语言**建立自己的概念空间，并在其中进行**组合、推理、决策、创造**。
> 

这不再是共形的扩张，而是**共形的深化**：从「形式的适配」走向「本体的生长」。

阶段二的终点是 SAM 3.1、Depth Anything v2、Florence-2、DINOv3——它们都非常强大，但都**不思考**。

阶段三将从这里开始。
