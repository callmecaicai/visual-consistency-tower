**主题**：区域/实例类稠密任务的架构收束——语义 / 实例 / 全景 / 视频对象分割等任务被抽象成「一组 mask 查询 + 每个查询一个标签」。EoMT 进一步说明，在强预训练 ViT 与适当 query 注入下，部分分割任务可以让 encoder 同时承担 backbone 与读出机制；这不是所有稠密视觉任务的终局，但它是阶段 II 晚期最强的统一形态之一。

## 1 · MaskFormer —— *Per-Pixel Classification is Not All You Need for Semantic Segmentation* (Cheng, Schwing, Kirillov, NeurIPS 2021)

**共形贡献**：语义分割不必做「逐像素分类」。改成「预测 N 个 mask + 每个 mask 一个类别」——这和实例分割用的是**同一个公式**。语义 / 实例分割第一次在架构上合流。

🔗 [https://arxiv.org/abs/2107.06278](https://arxiv.org/abs/2107.06278)

## 2 · Mask2Former —— *Masked-attention Mask Transformer for Universal Image Segmentation* (Cheng 等, CVPR 2022)

**共形贡献**：在 MaskFormer 基础上加 masked attention（query 只关注自己当前预测 mask 覆盖的区域）和高效多尺度 decoder。一个配置在语义 / 实例 / 全景三个任务上全部拿 SOTA——架构统一在经验上被证成。

🔗 [https://arxiv.org/abs/2112.01527](https://arxiv.org/abs/2112.01527)
