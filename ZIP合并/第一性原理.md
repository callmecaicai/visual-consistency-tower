# 第一性原理

**## 写给自己 · 本文件的用途**

这份文件不是给别人看的。是我，作为一个计算机视觉博士研究生，给自己的认知压缩算法。

CV 是一个每天几十篇 arxiv、每半年一次范式翻转的领域。我的认知带宽有限。没有一个一致的压缩字典，信息会溢出、会散、会变成焦虑和迷失。

「相容性 / 共形 / 残差 / 六轴」在我这里不是学术术语，是压缩算法的基本算子。

- **认知脚手架**：每读一篇新 paper 都能找到位置放
- **判断过滤器**：快速识别「这是哪一轴的展开、哪一层的残差」
- **定向工具**：知道自己的研究坐标在塔的哪一层
- **心理铠甲**：面对导师、reviewer、同行时有自己看领域的方式可依托
- **语言统一场**：检测 / 分割 / 3D / VLM 在我脑中是同一套语言

**武装自己不是做学术，是做自处。**

理论需要被扰动、被修正、甚至被推翻——但这是它在我身上工作之后自然发生的事，不是起点的动作。起点只有一个：**让我的脑子不散掉**。

**## 公理**

存在的最小本体论单位不是实体 (substance)，而是****相容性*\*** (compatibility)。

任何现象都是某一相容维度从****潜存态*\***向\**实现态****的****展开**。

此公理不由任何更基本的东西支撑。一旦被设定，所有后续概念皆从它推导，而非并立宣告。

- **--**

**## 原语**

理论仅依赖三个原语：

- ***相容性*\***——关系本体论的原语。不是实体属性，是关系结构之间可被协调的潜在可能。
- ***潜存 / 实现*\***——相容性的两种模态。潜存是尚未展开的相容可能；实现是已展开的相容事实。
- ***展开*\***——事件原语。相容性从潜存到实现的运动。不是主体对客体的塑造，是相容性对自身的兑现。
- **--**

**## 推论**

**### 共形**

展开在两个或多个关系结构之间的特例——****相容性的相互展开*\***。

**### 展开事件的四个结构条件**

任何展开要求：****被塑者、塑者、度量、动力*\***。缺任一，展开不成立。

非并立公设，是展开作为事件的逻辑必要条件。

**### 展开的六条轴**

相容性可沿六条轴展开：****材料、度量、动力、载体、时间尺度、主体性*\***。

非本体公设，是展开可变性的坐标。

**### 主体性**

- ***主体性强度 = 系统能将多少外部不相容度转化为内部展开压力的能力。*\***

连续谱，不是二值属性。

**### 反身**

- ***反身 = 主体性维度的极限。*\*** 当系统能将「度量自身的相容性」作为待展开的相容维度时，进入反身态。

反身不是终极问题，是贯穿性的自检——每一阶段皆可追问本阶段度量的来源。

**### 泛化**

- ***泛化 = 在已接触关系结构上展开的相容性，向共享同一潜存相容维度的未接触部分的自然外推。*\***

非神秘涌现，是展开机制的推论。

**### 涌现**

- ***涌现 = 某相容维度的信噪比穿越阈值，从潜存跃入实现。*\***

非新能力的诞生，是维度的现身。

**### 残差**

本层展开饱和，剩余压力必另寻通道。同一机制的两种尺度：

- ***技术残差*\***：剩余压力沿恒等通道传到下一层
- ***历史残差*\***：未展开维度定义下一阶段入口
- **--**

**## 元原则**

一、凡理论不从公理推导其概念者，其大厦不能久立。

二、本理论自身受此约束。

三、本理论的相容性需要被展开——在每一次新现象的到来中被扰动、修正、甚至被推翻。

四、可证伪性构成理论生长的承载通道。

- **--**

**## 哲学立场**

本公理与多个本体论传统——前定和谐、潜能到现实、过程哲学、差异与重复、种子现行——在最抽象层皆可对话，但不依附任一。

本理论是所有本体论传统在最抽象层的最大公约数，不是任何单一传统的翻译版。

- **--**
- ***相容不停，展开不止。*\***
