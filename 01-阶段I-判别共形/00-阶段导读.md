# 阶段 I · 判别共形导读

<aside>
🧭

**章首协议（接回第一原理层）**：本章按“稳定差异 → 闭合半径 → 出口残差”三问读取。

- **稳定差异**：整图可分的全局视觉差异，以及支撑分类成立的训练稳定性差异。
- **闭合半径**：把原始图像压到单一标签接口；判别共形第一次成为可复制、可扩展的工程体系。
- **出口残差**：标签闭合只在最后一维成立，空间内部、结构关系和开放语义都被压扁。

**阶段任务**：把「一张图 → 一个标签」这件事从工程偶然，推进成可复制、可扩展、可诊断的共形体系。

**阶段靶心**：用 ImageNet 时代的文献链，回答四个最基础的问题：

1. 数据与算力怎样把深度学习从实验室技巧推成时代主流。
2. 深度为什么一度是最有效的性能变量。
3. 训练为什么会坏，以及坏在什么地方。
4. 架构怎样从堆深走向分工，再走向残差闭合。

## 本阶段在整座大厦里的位置

如果说 [00-公理层/第一性原理](../00-公理层/第一性原理.md) 给出的是整座大厦的语言，那么阶段 I 做的就是第一层现实化：它把「相容 / 展开 / 残差」这些抽象词，第一次压到可测量的工程对象上。

阶段 I 的对象还很朴素：整张图只对应一个类别，输出空间仍然是离散标签。但正因为对象简单，这一层才能把整个视觉谱系最底部的承载结构打牢。这里立住的不是某一篇论文，而是四根长期有效的支柱：

- **材料**：ImageNet、GPU、批量训练，把“数据地基”真正铺开。
- **度量**：Top-1 / Top-5 使模型优劣第一次被大规模公共基准锁定。
- **动力**：初始化、激活、归一化、优化器共同定义训练是否稳定。
- **载体**：卷积、瓶颈、分支、残差把“表示如何流动”变成架构语言。

## 本阶段的残差链

阶段 I 不是一组并列论文，而是一条清晰的残差传递链：

1. **LeNet / DBN / ImageNet** 证明“深层表征 + 大数据”值得投入。
2. **AlexNet / ZFNet / NiN / VGG** 证明“更深 + 更大规模 + 更细粒度设计”是最短提分路径。
3. **激活 / 初始化 / 正则** 说明深度不是免费午餐，训练稳定性本身需要一整套动力学工艺。
4. **Inception / BatchNorm** 说明简单堆深会撞墙，必须引入宽度、分支和归一化。
5. **ResNet / 优化器革命** 最终把“超深网络可训练”这件事从例外变成常态。

这条链闭合之后，阶段 I 留下的真正出口残差只有一条：

> **共形已经能够稳定穿过很深的网络，但它仍然只在“整张图 ↔ 一个标签”的粗粒度上工作。**

这正是阶段 II 必须转向“每一像素、每一位置、每一种结构化输出”的原因。
