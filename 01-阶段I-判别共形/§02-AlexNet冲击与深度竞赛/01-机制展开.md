## §1 · AlexNet：ImageNet 为什么会在 2012 年突然被打穿

AlexNet 最常被误读成“只是把网络做大了”。更准确地说，它第一次把三件原本分散的东西压进同一个训练管道：

- **算力并行**：两张 GPU 分摊模型与特征图，证明深网可以和工业级硬件协同演化。
- **非线性与正则工艺**：ReLU、Dropout、数据增强不再是零散技巧，而是完整训练制度。
- **数据集规模**：ImageNet 让“深层表征是否值得学”第一次能在百万量级样本上被验证。

AlexNet 的历史意义不在于它的 11×11 卷积今天仍最优，而在于它确立了一个时代命题：

> **性能跃迁来自“数据规模 × 算力供给 × 可训练架构”的乘法，而不是单一结构灵感。**

这条命题一旦成立，后续所有工作都会自然滑向“更深、更稳、更省”的方向。

## §2 · ZFNet：黑箱第一次变成可诊断对象

AlexNet 赢了，但为什么赢、哪里还粗糙，在 2012 年其实没人说得足够清楚。ZFNet 的关键贡献，不是又提了一个冠军模型，而是让卷积网络第一次有了“可看”的内部几何。

反卷积可视化做了两件事：

1. 把不同层对输入图像的响应区域投回像素平面，让研究者看到“边缘 → 纹理 → 部件”的层级展开。
2. 直接暴露结构缺陷。ZFNet 发现 AlexNet 首层大步长卷积过早丢失了细节，于是改小卷积核和步长，性能随之上升。

从这一刻起，架构设计不再只是“试一试能不能涨点”，而是开始拥有几何诊断的反馈回路。也就是说，**ZFNet 把深网从工程偶然推向了可解释调参**。

## §3 · NiN：微结构革命如何提前预埋未来十年

NiN 的表面改动不大，但它提前交出了两件影响极深的基础零件。

### 3.1 1×1 卷积

1×1 卷积的作用不是“空间上看得更广”，而是**在不改变空间拓扑的前提下重塑通道空间**。这意味着：

- 空间位置关系保持不变。
- 通道之间的线性组合变成可学习对象。
- “先压缩 / 再处理 / 再展开”的瓶颈设计开始成为可能。

从后来的 Inception、ResNet bottleneck、MobileNet pointwise conv，到 Transformer 的逐 token MLP，本质上都在重复这条语法：**先在不打乱空间结构的前提下换坐标，再决定怎样计算。**

### 3.2 全局平均池化

GAP 则是另一种更隐蔽的革命。它让网络末端不再依赖巨大全连接层来“收尾”，而是把“类别证据”直接理解为全图上的空间平均。这一步的意义是：

- 参数量骤减，减轻过拟合。
- 分类头从“记忆器”转向“统计读出器”。
- “卷积主干 + 轻量读出”这一现代范式第一次被明确写出来。

NiN 因此不只是 AlexNet 时代的配角，而是后续所有微结构共形设计的源头之一。

## §4 · VGG：把深度本身提炼成最强变量

如果说 AlexNet 证明“深网能赢”，VGG 证明的则是更尖锐的话：

> **在固定数据集和评价制度下，深度本身就是最可重复、最可靠的提分变量。**

VGG 的策略看似朴素到近乎笨拙：只用 3×3 卷积，不玩花样，单纯把层数往上堆。但它之所以重要，恰恰因为它把问题压缩到了最少自由度：

- 两个连续 3×3 的感受野等价于一个 5×5，但参数更少、非线性更多。
- 小卷积核的重复堆叠使“更深”从口号变成可工程化的设计原则。
- 当结构足够统一时，深度的作用被单独凸显出来，方法论开始可迁移。

VGG 也因此把阶段 I 的主旋律彻底钉死了：**只要训练还能撑住，继续加深几乎总是对的。**

## §5 · 四篇合起来，真正建立了什么

把这四篇并排看，它们完成的不是四个独立技巧，而是一条逐步收束的方法链：

- **AlexNet** 证明规模化深网可行。
- **ZFNet** 证明深网可诊断。
- **NiN** 证明深网内部还能继续做微结构优化。
- **VGG** 证明深度本身值得被单独当作战略变量推进。

于是，深度学习在 2012-2014 之间完成了一次关键身份变化：它不再只是“某个模型赢了比赛”，而是变成了**一套可复制的架构研究程序**。

# 第二部分 · 本页合力的残差

## §6 · 面向下一页的入口形状

到 VGG-19 为止，结论已经清楚：更深几乎总是更强，但“单纯堆深”也开始逼近工程极限。

- 参数与显存压力快速膨胀。
- 梯度传播越来越脆弱。
- 结构仍然过于单线，宽度与分支几乎没有被系统利用。

这意味着下一步不能只是“更深一点”，而必须改写深度本身的组织方式。于是 [阶段 I · §4 · Inception 族与归一化（GoogLeNet / BatchNorm / Inception v3 / Inception-v4）](..\§04-Inception族与归一化.md) 要引入**宽度 + 稀疏连接 + 归一化**三元方案，把“深度”从单线堆叠推进到分工协同。
