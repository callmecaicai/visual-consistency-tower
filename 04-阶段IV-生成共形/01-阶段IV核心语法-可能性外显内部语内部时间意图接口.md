## 一、阶段 IV 的纯存在态

阶段 IV 的纯存在态不是“模型会生成”，而是：

> 视觉世界不再只是等待被识别、被分割、被命名；它开始能够从系统内部被生产出来。

最小公式是：

```text
(z, c, τ, D) -> x_hat
```

其中：

| 符号 | 含义 |
|---|---|
| `z` | 内部潜变量、噪声、token 或表征 |
| `c` | 条件、意图、文本、结构约束或参考输入 |
| `τ` | 生成轨迹、去噪时间、采样路径 |
| `D` | 生成动力学 |
| `x_hat` | 被外显出来的可见图像样本 |

阶段 IV 的全部技术史，都是为了让这条路径成立：

```text
内部可能性 -> 可见样本
```

阶段 IV 的全部残差，也都来自这条路径的成功：

```text
能外显，不等于能持有。
```

## 二、外显生产主体性的边界

阶段 IV 的主体性必须限定为“外显生产主体性”。它意味着：

1. 系统能从内部变量出发产生可见样本。
2. 系统能在某种动力过程中修正样本。
3. 系统能接受条件或意图约束。
4. 系统能把内部状态外化给观者。

它还不意味着：

1. 系统持有世界状态。
2. 系统理解对象因果。
3. 系统能自我修订度量。
4. 系统拥有行动后果。

因此本阶段的主体不是完整主体，而是：

```text
IV = 外显生产主体
V  = 内部持有主体
```

## 三、生成闭合的五个判据

一个生成系统在阶段 IV 中要形成非塌缩闭合，至少要回答五个问题：

| 判据 | 问题 | 典型失败 |
|---|---|---|
| 可能性 | `x_hat` 是否来自可学习的图像可能性空间？ | 样本只是记忆或随机拼接 |
| 可信性 | `x_hat` 是否经受外部判别、人类感知或度量抵抗？ | 画面锐利但整体不可信 |
| 多样性 | 模型是否覆盖分布，而不是只会少数模式？ | mode collapse、风格单调 |
| 可控性 | 条件 `c` 是否稳定改变应改变部分并保持不变量？ | prompt 跟随弱、编辑污染全局 |
| 轨迹性 | 生成是否拥有可解释、可压缩、可加速的动力路径？ | 只能慢采样，不能分析或控制过程 |

这五个判据不是单一分数，而是阶段 IV 的结构化半径。一个模型可以在画质上很强，却在可控性、身份保持、结构一致性或采样轨迹上失败。

## 四、阶段 IV 的六种闭合

阶段 IV 的章节不按年表排列，而按闭合对象展开：

| 闭合类型 | 解决的问题 | 对应章节 | 出口残差 |
|---|---|---|---|
| 可能性闭合 | 什么样的图像可能成立？ | `§31` 密度、能量、自回归 | 可描述分布，但采样、控制、速度困难 |
| 可信性闭合 | 生成样本如何通过外部判别？ | `§32` GAN | 逼真但不稳，覆盖不足，内部语弱 |
| 内部语闭合 | 生成系统内部如何组织自己？ | `§33` VAE、VQ-VAE、Flow、MaskGIT | 有坐标，但质量、时间、控制未统一 |
| 显形轨迹闭合 | 样本如何逐步显形？ | `§34` Diffusion、Score、Flow | 动力成熟，但条件接口未统一 |
| 意图/工艺闭合 | 如何按人类意图、结构约束和产品需求生成？ | `§35-§36` 文本、ControlNet、DiT、蒸馏、编辑 | 产出成熟，但状态持有仍未成立 |
| 反身表征闭合 | 生成内部状态能否成为理解载体？ | `§37` MAE、DIFT、REPA | 送入阶段 V |

这张表的纪律是：同一方法可以跨阶段出现，但主归属取决于它主要扩张了哪一种闭合。

## 五、生成家族对照表

阶段 IV 的模型家族不按“谁更先进”排序，而按它们回答的闭合问题区分：

| 家族 | 最小问题 | 主要闭合 | 典型残差 |
|---|---|---|---|
| EBM / RBM / DBN | 什么图像可能成立？ | 可能性地形 | `Z` 难算、采样困难 |
| Autoregressive | 图像能否被显式顺序写出？ | 显式似然 | 慢、顺序僵硬 |
| GAN | 样本能否通过判别边界？ | 外显可信性 | 博弈不稳、模式崩溃 |
| VAE / β-VAE | 内部潜变量如何可采样？ | 连续内部语 | 样本发糊、语义可控弱 |
| VQ-VAE / MaskGIT | 内部语能否离散组合？ | 离散内部语 | 码本语义不稳、依赖解码器 |
| Flow | 图像和潜变量能否严格可逆？ | 可逆坐标 | 架构受限、成本高 |
| Diffusion / LDM | 图像如何沿轨迹显形？ | 显形轨迹 | 慢、依赖采样与条件工艺 |
| DiT / Flow Matching | 生成动力如何规模化和连续化？ | 生产制度 / 轨迹系统化 | 工艺复杂，仍需评价账本 |

## 六、公式旋钮表

| 旋钮 | 所在机制 | 它控制什么 | 典型风险 |
|---|---|---|---|
| `z` | VAE / GAN / Flow | 内部变化入口 | 只是噪声，不等于内部语 |
| `β` | β-VAE / DDPM | KL 权重或噪声调度 | 过大压细节，过小秩序弱 |
| `Z` | EBM | 概率归一化 | 难算导致训练和采样困难 |
| `D` | GAN | 判别边界 | 太强梯度枯竭，太弱反馈无效 |
| KL | VAE | posterior 与 prior 的距离 | 过强会 posterior collapse |
| `ε` | DDPM | 噪声目标 | 学错噪声就学错去噪方向 |
| `t` | Diffusion | 生成动力学时间 | 时间调度不当会影响质量和速度 |
| `w` / `s` | CFG | 条件强度 | 过强僵硬，过弱不听 prompt |
| `λ` | WGAN-GP / REPA | 正则或对齐权重 | 过强压制自由，过弱约束无效 |

| 等级 | 读者任务 | 本阶段公式 |

## 八、阶段 IV 的非塌缩条件

阶段 IV 的闭合不能只看“能不能出图”。以下情况都不是完整闭合：

- 高质量但单调：可信性强，多样性塌缩。
- 听 prompt 但结构崩：意图接口强，空间状态弱。
- 能编辑但身份漂移：局部控制强，不变量持有弱。
- FID 好但不可用：分布相似强，任务约束弱。
- 图像漂亮但关系错：外显产物强，世界状态弱。

因此阶段 IV 的核心判据应写成：

> 能生成是理解的强外显判据，但不是理解的最终本体。

## 九、阶段 IV 的自我否定

阶段 IV 的自我否定不是“生成模型还不够强”，而是：

```text
生成越强，越暴露最终样本不是理解本身。
```

它分四层返回：

| 层级 | 自我否定 | 指向 |
|---|---|---|
| 样本可信性 | 逼真样本可以没有稳定世界状态 | 结构、手指、文字、身份、物体持久性 |
| prompt 可控性 | 语言约束不能保证空间、数量、关系和因果一致 | 计数、左右、多主体关系 |
| 编辑控制 | 可控编辑要求知道什么不该改 | 不变量持有 |
| 中间状态反身 | 中间特征比最终图像更像理解载体 | 阶段 V 的表征共形 |

最终收束为一句话：

> 阶段 IV 的失败不是不会生成；阶段 IV 的失败来自它太会生成，以至于人们发现最终样本不是理解本身。
