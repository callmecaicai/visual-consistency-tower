# 阶段 IV · 数学最小词典

类型: 理论/术语入口
阶段: IV · 生成共形
共形维度: 材料, 动力, 度量
关键贡献: 给不熟生成模型公式的读者一个最低限度词典。这里不替代数学推导，只保证读者能看懂阶段 IV 的主线判断。

## 一、概率与能量

| 词 | 最小解释 | 本阶段位置 |
|---|---|---|
| `p(x)` | 图像 `x` 在某个分布下的概率或密度 | `§31` 可能性闭合 |
| `E(x)` | 能量，越低通常表示越可能 | `§31` EBM |
| `Z` | 归一化常数，让所有概率加起来合法；常常难算 | `§31` 的第一大病理 |
| score | `∇_x log p(x)`，表示往哪里移动更像数据 | `§31 -> §34` 的桥 |
| likelihood | 模型给观测分配的概率 | `§31`、Flow、评价账本 |

## 二、潜变量与变分

| 词 | 最小解释 | 本阶段位置 |
|---|---|---|
| `z` | 内部潜变量、噪声、token 或表征入口 | 阶段 IV 总公式 |
| prior | 先验，通常是生成前假定的简单分布 | VAE、Flow、扩散 |
| posterior | 后验，看到 `x` 后对 `z` 的推断 | VAE |
| KL | 两个分布之间的差异度量 | VAE / β-VAE |
| ELBO | `log p(x)` 的可优化下界 | VAE |
| reparameterization | 把随机采样改写成可微参数加外部噪声 | VAE |

## 三、离散内部语

| 词 | 最小解释 | 本阶段位置 |
|---|---|---|
| codebook | 一组离散视觉词汇向量 | VQ-VAE |
| commitment | 让编码器输出愿意贴近码本的约束 | VQ-VAE |
| token | 可被序列化或组合的离散单位 | VQ-VAE、MaskGIT、DiT |
| mask-and-fill | 遮住一批 token 再逐步补全 | MaskGIT / Muse |

## 四、可逆流与速度场

| 词 | 最小解释 | 本阶段位置 |
|---|---|---|
| log-det | 可逆变换改变体积时需要补偿的项 | Flow |
| Jacobian | 描述变换局部如何拉伸或压缩空间的矩阵 | Flow |
| velocity | 连续流里从一个状态移动到另一个状态的速度 | Rectified Flow / Flow Matching |
| transport path | 从噪声分布到数据分布的路径 | Flow Matching |

## 五、扩散

| 词 | 最小解释 | 本阶段位置 |
|---|---|---|
| `x_0` | 原始干净图像 | DDPM |
| `x_t` | 第 `t` 步加噪后的图像 | DDPM |
| `ε` | 真实加入的噪声 | DDPM |
| `ε_θ` | 网络预测的噪声 | DDPM |
| `β_t` / `σ` | 噪声强度或噪声调度参数 | DDPM / EDM |
| sampler | 把训练好的去噪模型转成实际生成轨迹的数值算法 | DDIM、DPM++、UniPC |
| distillation | 用教师模型或长轨迹训练少步学生模型 | LCM、Consistency、DMD |
| guidance | 用条件信号推采样方向 | CFG、classifier guidance |

## 六、条件接口

| 词 | 最小解释 | 本阶段位置 |
|---|---|---|
| `c` | 条件、意图、文本、结构约束或参考输入 | 阶段 IV 总公式 |
| CFG scale | 条件推动采样的强度 | `§34`、`§35` |
| cross-attention | 图像 token 从文本 token 读取信息 | `§35` |
| negative prompt | 用文本描述“不想要什么” | `§35` |
| ControlNet | 把边缘、深度、姿态等结构条件接入扩散 | `§35`、`§36` |
| LoRA | 低秩增量参数，用于快速适配风格或身份 | `§36` |

## 七、反身桥

| 词 | 最小解释 | 本阶段位置 |
|---|---|---|
| intermediate feature | 生成模型中间层的表征 | `§37` |
| feature usefulness | 中间特征能帮下游任务 | `§37` 的 `G1` |
| representation transfer | 中间特征能跨接口迁移 | `§37` 的 `G2` |
| state holding | 特征稳定持有对象、几何、遮挡等状态 | 阶段 V 门槛 |
| REPA | 用外部强视觉表征校准生成器中间状态 | `§37` |

## 八、最短记忆

阶段 IV 的数学只要先记住四句话：

1. EBM 问“什么图像可能成立”。
2. VAE / VQ-VAE / Flow 问“内部语怎么组织”。
3. Diffusion 问“图像如何沿时间显形”。
4. CFG / cross-attention / ControlNet 问“意图如何进入生成轨迹”。
