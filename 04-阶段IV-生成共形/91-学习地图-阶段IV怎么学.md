# 阶段 IV · 学习地图：这一层应该怎么学

本文件服务两类读者：

- 人类读者：先抓主线，再看公式，不要一上来把自己淹死在模型名里。
- Agent：先定位章节，再按深度读取，不要把 `§31-§37` 一次性全吞。

## 一、先记住阶段 IV 的一句话

阶段 IV 研究的不是“生成模型编年史”，而是这条认识论翻转：

> 视觉从“能被判别”翻到“能外显生产可见样本”。

再压短一点就是：

> 阶段 IV 的主命题是：能生成是理解的强外显判据。

但它在最后一章又会亲手收紧这条公设：能外显，不等于能持有。这为阶段 V 铺路。

## 二、七章接力线

1. `§31` 前夜：生成问题早就被提出，但一直没形成稳定外显生产闭合。
2. `§32` 外显可信性：GAN 让样本第一次通过共同学习的判别边界获得可信性。
3. `§33` 内部语：潜空间工程让生成系统有了可组织的内部坐标。
4. `§34` 内部时间：扩散把生成从一次映射改写成持续显形。
5. `§35` 意图接口：语言成为最通用的公共条件接口，结构条件补足约束。
6. `§36` 生产制度：扩散进入完整工业栈，并暴露评价账本。
7. `§37` 反身桥：生成器暴露出自己更高价值在中间表征，而不只是最终样本。

## 三、第一次阅读顺序

如果你对生成模型还不熟，建议顺序如下：

1. 先读 [00-总纲](00-总纲-从判别到生成的认识论翻转.md)
2. 再读 [01-阶段IV核心语法](01-阶段IV核心语法-可能性外显内部语内部时间意图接口.md)
3. 用 [02-生成模型公式地图](02-生成模型公式地图.md)、[03-参数敏感度账本](03-参数敏感度账本.md)、[04-数学最小词典](04-数学最小词典.md) 建立公式底座
4. 再读 [§34-扩散革命与内部时间/00-定位卡](§34-扩散革命与内部时间/00-定位卡.md)
5. 然后读 [§34-扩散革命与内部时间/01-机制展开](§34-扩散革命与内部时间/01-机制展开.md)
6. 回头补 `§31 -> §32 -> §33`
7. 再读 `§35 -> §36`
8. 最后读 `§37` 和 [结语](结语-为观者产出与为自己持有.md)

原因很简单：

- 扩散是阶段 IV 的主轴。
- 先抓主轴，再回头看前夜、可信性闭合、内部语，会容易很多。

## 四、如果你怕公式，先抓这几条

先用七个问题定位章节：

| 问题 | 对应章节 | 无公式版一句话 |
|---|---|---|
| 什么样的图像可能成立？ | `§31` | 生成首先要知道什么样的图像可能成立。 |
| 生成样本如何获得可信性？ | `§32` | GAN 让生成样本通过对手的判别获得可信性。 |
| 生成系统内部如何组织变化？ | `§33` | 潜空间让生成系统有了内部坐标和内部词汇。 |
| 图像如何逐步显形？ | `§34` | 扩散把生成从一次吐图改成逐步显形。 |
| 人类意图如何进入生成？ | `§35` | 文本和结构条件让生成系统听懂“想要什么”和“必须怎样”。 |
| 生成如何成为生产制度？ | `§36` | 生成不只是模型，而是一整套生产工艺。 |
| 为什么最终样本不是终点？ | `§37` | 最终图像可能不是理解本身，中间状态才更像理解载体。 |

公式按三层读：

| 等级 | 目标 | 代表内容 |
|---|---|---|
| `F0` | 先能用一句话讲机制 | GAN = `G` 造图 `D` 挑错；VAE = `z` 重建又规整；Diffusion = 猜噪声去噪；CFG = 推强条件；LDM = latent 扩散 |
| `F1` | 看懂主线公式在问什么 | GAN minimax；VAE ELBO；DDPM noise prediction；CFG；Flow change of variables |
| `F2` | 二遍阅读或专题阅读 | Score-SDE；Wasserstein dual；EDM 参数化；Consistency；Rectified Flow / Flow Matching；DiT scaling law |

读公式时不要求一步到位推导。最低要求是能回答：

```text
这个公式在问什么？
哪个参数是旋钮？
旋钮变大或变小会怎样？
它服务哪一种生成闭合？
```

### 4.1 `§31`

- 能量模型：`p(x) = exp(-E(x)) / Z`
- score：`s(x) = \nabla_x \log p(x)`
- 只要懂“高度难算，方向更关键”，就够了。

### 4.2 `§32`

- 原始 GAN：`min_G max_D ...`
- WGAN：把真假分布差别改写成更可优化的距离
- 只要懂“GAN 让动力来自对手”，就够了。

### 4.3 `§33`

- VAE 的 ELBO
- VQ-VAE 的离散码本
- Flow 的 change of variables
- 只要懂“潜空间 = 生成系统内部语”，就够了。

### 4.4 `§34`

- `q(x_t | x_0)`：前向加噪
- `\|\epsilon - \epsilon_\theta(x_t, t)\|^2`：噪声预测
- CFG：条件如何推着采样走
- 只要懂“扩散 = 内部时间”，就够了。

### 4.5 `§35`

- CLIP / T5：文本编码器
- cross-attention：文本怎样进网络
- CFG / prompt / adapter：文本怎样真正变得可用

### 4.6 `§36`

- 骨干：U-Net → DiT → MMDiT
- 速度：DDIM / DPM++ / UniPC / Consistency / 蒸馏
- 控制：ControlNet / 编辑 / 个性化

### 4.7 `§37`

- MAE / BEiT：重建也能学出强表征
- DIFT / REPA：生成器的中间特征本身就是好表征
- 只要懂“中间状态比最终样本更像理解”，就够了。

## 五、整层最容易混淆的三条边界

### 5.1 `III vs IV`

- 看图说话、图像问答、理解侧 VLM：仍在 III。
- 从零生成图像、编辑图像、条件生成：进入 IV。

### 5.2 `IV vs V`

- 目标仍是像素、潜变量、像素级 token 的恢复：在 IV。
- 目标变成抽象状态、表征向量、世界状态预测：进入 V。

### 5.3 “有语言”不等于“是语言阶段”

- 阶段 III 用语言锚定视觉语义。
- 阶段 IV 用语言表达生成意图。

两者都用语言，但主语不同。

## 七、这一层学完应该得到什么

学完阶段 IV，不应该只记住一串模型名。应该得到的是这条链：

> 生成问题被提出，样本获得外显可信性，系统有了内部语，内部语有了生成动力学时间，时间接上意图接口，接口长成生产制度，而生产制度最后又把自己暴露为表征机器。

如果这条链清楚了，阶段 IV 就算真正学会了。
