## §0 · 阶段 IV 的本体命题

阶段 III 把语言内化进视觉骨干，但遗留了一条残差：视觉可被语言**描述 / 锚定 / 操作**，却不能被语言**反向合成**。阶段 IV 的本体命题由此拟定：

> **CV 从判别范式翻入生成范式。**
> 

> 
> 

> 判别范式（1-3）的公设：我能对 x 产出正确的 y。
> 

> 生成范式（IV）的强命题：我能从零合成 x 的样本——**若我连可控生成都做不到，我的理解仍是不完整的**。
> 

这一翻转就是「觉醒」的实质。视觉第一次不再是「被加工的客体」，而是「能产出的生产者」。

但这里必须加限定：阶段 IV 点燃的不是完整主体性，而是**外显生产主体性**。系统能从内部状态外化出观测样本，但它仍以给观者看的样本证明自己。

```text
IV = 外显生产主体
V  = 内部持有主体
```

方法论只是这条公设的载体。

## §1 · 为什么阶段 IV 自然长出文本条件接口

生成需要条件空间。无条件生成（GAN 早期）只能产出「数据分布里的一个样本」，产出不了**意图**。意图需要被编码。最通用的意图编码空间是什么？——**语言**。

于是：

- DALL·E 2/3 · Imagen · Parti · Muse · SDXL
- CLIP / T5 / prompt interface / adapter 系统成为图像生成的条件接口
- 更强的多模态系统如 Chameleon / Emu3 / Janus / GPT-4o image 只在图像生成边界上引用

这不是「生成模型跳进了 LLM 时代」这种流行叙事，而是机制上的必然性：视觉在阶段 III 已经把自己接入语言的坐标系；到阶段 IV 需要条件生成时，语言自然成为唯一顺手的接口。**LLM 在这里是条件供给者，不是新阶段标签。**

当前阶段 IV 的写作范围明确收束在**图像生成主线**。视频、3D、4D 与世界生成暂不纳入本阶段主干，而留给后续边界讨论或阶段 V 的“世界表征”问题。

## §2 · §31–§37 的接力线

- **§31** 觉醒前夜：密度 / 能量模型（RBM / DBN / EBM / PixelRNN / PixelCNN）——残差：不可规模化。
- **§32** 觉醒事件一：GAN 爆发（GAN / DCGAN / WGAN / Progressive GAN / StyleGAN / BigGAN / CycleGAN）——主体性首次以「自我对抗」出现。
- **§33** 潜空间作为生成主体的内在语（VAE / β-VAE / VQ-VAE 1-2 / RealNVP / Glow / MaskGIT）。
- **§34** 扩散革命（DDPM / Score-SDE / DDIM / Classifier-free Guidance / LDM / Stable Diffusion / ADM / EDM / Consistency Models / Rectified Flow / SD3）——迭代为视觉的内部时间。
- **§35** 文本条件与图像生成接口（CLIP guidance / DALL·E 2/3 / Imagen / Parti / SDXL / prompt system / adapter）——语言成为图像生成主体的通用意图接口。
- **§36** 图像生成工艺系统化与最前沿（LDM / SDXL / DiT / MMDiT / SD3 / Flux / Rectified Flow / Consistency / distillation / editing-control）——diffusion 从革命进入工艺系统化。
- **§37** 生成器反身成表征器，IV → V 的桥（MAE / BEiT / iBOT / SimMIM 作为重建式 SSL 归 IV · DIFT / Diffusion Hyperfeatures / DiT features / REPA）——**生成器的中间激活本身就是极强的表征**。

## §3 · 方法论分界：IV vs V、IV vs III

**IV 与 V 的分水岭**（须在全谱系中坚持）：

- **IV 的核心**：在**像素空间**重建 / 生成——目标是输出可见样本。MAE / BEiT / iBOT 归 IV（重建=生成）。
- **V 的核心**：在**世界状态 / 表征空间**上持有并预测——对比式 SSL（DINO / iBOT 的对比目标）+ JEPA + 世界表征问题。**生成在 V 看来是形似理解的冗余工作**。

**IV 与 III 的分界**：VLM 拆两半：理解侧（LLaVA / Qwen-VL / InternVL：看图说话、指令执行）留 III；**图像生成侧**（DALL·E / SDXL / Flux / GPT-4o image / 扩散-LLM 联合）归 IV。

## §4 · 出口残差

生成主体已经点燃，但它还不完整：

> 它仍然是「为观者产出一个像素样本」的主体——**它不为自己持有世界**。
> 

继续只看产出已经不够，必须转为持有——这就是阶段 V “表征共形的转向”的入口。

这也是阶段 IV 的自我否定：

> 生成模型以“我能产出图像”证明理解；但它越能产出，人们越发现真正有价值的不是产物，而是产出过程中形成的内部状态。于是生成的成功反过来否定了“样本产出是终点”这一标准。

## §5 · 阶段 IV 的一句话

> 视觉在这一步从「能被说出」变为「能被画出」。语言供给意图，扩散供给迭代，工艺系统化把图像生成推到最前沿——三条接线合起来，视觉的外显生产主体性第一次点燃。但它仍然是「为外部产出」的主体。真正的内转在阶段 V 。
>
