## §0 · 阶段 IV 的本体命题

阶段 III 把语言内化进视觉骨干，但遗留了一条残差：视觉可被语言**描述 / 锚定 / 操作**，却不能被语言**反向合成**。阶段 IV 的本体命题由此拟定：

> **CV 从判别范式翻入外显生产范式。**

阶段 IV 的最小公式不是 `x -> y`，也不是 `image <-> text`，而是：

```text
(z, c, τ, D) -> x_hat
```

其中 `z` 是内部潜变量、噪声、token 或表征，`c` 是条件或意图，`τ` 是生成轨迹，`D` 是生成动力学，`x_hat` 是被外显出来的可见样本。

判别范式（I-III）的公设是：

```text
我能对 x 产出正确的 y。
```

外显生产范式（IV）的强命题是：

```text
我能从内部状态外显出 x_hat。
```

这不是理解的最终本体，而是理解的强外显判据：若一个系统完全不能可控地外显视觉样本，它的视觉理解仍然是不完整的；但即使它能外显样本，也不自动等于它持有世界。

这一翻转就是「觉醒」的实质。视觉第一次不再只是「被加工的客体」，而是「能把内部可能性外显为可见样本的生产者」。

但这里必须加限定：阶段 IV 成立的不是完整主体性，而是**外显生产主体性**。系统能从内部状态外化出观测样本，但它仍以给观者看的样本证明自己。

```text
IV = 外显生产主体
V  = 内部持有主体
```

方法论只是这条公设的载体。

## §1 · 为什么阶段 IV 自然长出文本条件接口

生成需要条件空间。无条件生成（GAN 早期）只能产出「数据分布里的一个样本」，产出不了**意图**。意图需要被编码。最通用的意图编码空间是什么？——**语言**。

于是：

- DALL·E 2/3 · Imagen · Parti · Muse · SDXL
- CLIP / T5 / prompt interface / adapter 系统成为图像生成的条件接口
- 更强的多模态系统如 Chameleon / Emu3 / Janus / GPT-4o image 只在图像生成边界上引用

这不是「生成模型跳进了 LLM 时代」这种流行叙事，而是机制上的连续性：视觉在阶段 III 已经把自己接入语言的坐标系；到阶段 IV 需要条件生成时，语言成为最通用、最公共、最容易规模化的人类意图接口。它不是唯一条件接口：mask、depth、pose、edge、segmentation、reference image、style image、identity / LoRA、ControlNet 与 IP-Adapter 等结构条件同样重要。**LLM 在这里是条件供给者或 prompt 组织者，不是新阶段标签。**

当前阶段 IV 的写作范围明确收束在**图像生成主线**。视频、3D、4D 与世界生成暂不纳入本阶段主干，而留给后续边界讨论或阶段 V 的“世界表征”问题。

## §2 · §31–§37 的接力线

- **01 核心语法**：定义 `(z,c,τ,D) -> x_hat`，给出生成闭合五判据与非塌缩条件。
- **02 公式地图**：把 GAN、VAE、DDPM、CFG、REPA 等核心公式改写成“公式在问什么”的读者契约。
- **03 参数账本**：说明每个关键旋钮变大 / 变小时，哪种生成闭合被强化，哪种闭合被牺牲。
- **04 数学词典**：给 `p(x)`、`Z`、ELBO、score、`x_t`、CFG scale、sampler 等术语建立最低门槛。
- **§31** 可能性闭合：密度 / 能量模型（RBM / DBN / EBM / PixelRNN / PixelCNN）——残差：可能性地形已出现，但采样、控制、速度困难。
- **§32** 外显可信性闭合：GAN 爆发（GAN / DCGAN / WGAN / Progressive GAN / StyleGAN / BigGAN / CycleGAN）——样本通过共同学习的判别边界获得外显可信性。
- **§33** 内部语闭合：潜空间作为生成系统的内部语（VAE / β-VAE / VQ-VAE 1-2 / RealNVP / Glow / MaskGIT）。
- **§34** 显形轨迹闭合：扩散革命（DDPM / Score-SDE / DDIM / Classifier-free Guidance / LDM / Stable Diffusion / ADM / EDM / Consistency Models / Rectified Flow / SD3）——迭代为视觉的生成动力学时间。
- **§35** 意图接口闭合：文本条件与图像生成接口（CLIP guidance / DALL·E 2/3 / Imagen / Parti / SDXL / prompt system / adapter）——语言成为最通用的公共意图接口，结构条件继续补足约束。
- **§36** 生产制度闭合：图像生成工艺系统化与最前沿（LDM / SDXL / DiT / MMDiT / SD3 / Flux / Rectified Flow / Consistency / distillation / editing-control）——diffusion 从革命进入生产制度。
- **§37** 反身表征闭合，IV → V 的桥（MAE / BEiT / SimMIM；iBOT 作为桥接节点；DIFT / Diffusion Hyperfeatures / DiT features / REPA）——**生成器的中间激活开始比最终样本更像理解载体**。

## §2.5 · 七个问题与两层阅读法

阶段 IV 以后不按模型名背诵，而按七个问题进入：

| 问题 | 对应章节 | 无公式版一句话 |
|---|---|---|
| 什么样的图像可能成立？ | `§31` | 生成首先要知道图像可能性空间长什么样。 |
| 生成样本如何获得可信性？ | `§32` | GAN 让生成样本通过对手的判别获得可信性。 |
| 生成系统内部如何组织变化？ | `§33` | 潜空间让生成系统有了内部坐标和内部词汇。 |
| 图像如何逐步显形？ | `§34` | 扩散把生成从一次吐图改成逐步显形。 |
| 人类意图如何进入生成？ | `§35` | 文本表达想要什么，结构条件规定必须怎样。 |
| 生成如何成为生产制度？ | `§36` | 生成不只是模型，而是一整套生产工艺。 |
| 为什么最终样本不是终点？ | `§37` | 中间状态开始比最终图像更像理解载体。 |

因此，本阶段有两层阅读结构：

| 层级 | 目标 | 推荐入口 |
|---|---|---|
| 无公式层 | 先抓七个问题和七句答案 | `91-学习地图`、本总纲、各章学习抓手 |
| 公式层 | 再看每个公式如何规定闭合制度 | `02-生成模型公式地图`、`03-参数敏感度账本`、`04-数学最小词典` |

## §3 · 方法论分界：IV vs V、IV vs III

**IV 与 V 的分水岭**（须在全谱系中坚持）：

- **IV 的核心**：在**像素空间、潜空间或像素级 token 空间**重建 / 生成——目标是输出可见样本或恢复可见结构。MAE / BEiT / SimMIM 归 IV；iBOT 的 MIM 侧可作为 IV 桥接节点。
- **V 的核心**：在**世界状态 / 表征空间**上持有并预测——对比式 SSL（DINO / iBOT 的自蒸馏和 dense token 侧）+ JEPA + 世界表征问题。生成在 V 看来不是无价值，而是不再是理解的最终判据。

**IV 与 III 的分界**：VLM 拆两半：理解侧（LLaVA / Qwen-VL / InternVL：看图说话、指令执行）留 III；**图像生成侧**（DALL·E / SDXL / Flux / GPT-4o image / 扩散-LLM 联合）归 IV。

## §4 · 出口残差

外显生产主体已经成立，但它还不完整：

> 它仍然是「为观者产出一个像素样本」的主体——**它不为自己持有世界**。
> 

继续只看产出已经不够，必须转为持有——这就是阶段 V “表征共形的转向”的入口。

这也是阶段 IV 的自我否定：

> 生成模型以“我能产出图像”证明理解；但它越能产出，人们越发现真正有价值的不是产物，而是产出过程中形成的内部状态。于是生成的成功反过来否定了“样本产出是终点”这一标准。

## §5 · 阶段 IV 的一句话

> 视觉在这一步从「能被说出」变为「能被画出」。语言供给意图，扩散供给迭代，工艺系统化把图像生成推到最前沿——三条接线合起来，视觉的外显生产主体性第一次成立。但它仍然是「为外部产出」的主体。真正的内转在阶段 V。
>
