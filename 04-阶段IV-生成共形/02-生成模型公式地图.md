# 阶段 IV · 生成模型公式地图

类型: 理论/公式入口
阶段: IV · 生成共形
共形维度: 动力, 度量, 时间尺度
关键贡献: 把阶段 IV 的核心公式从“数学障碍”改写为“闭合制度的契约”。本页不追求推导完备，而是让读者知道每个公式在问什么、哪个参数是旋钮、它服务哪一种生成闭合。
残差 / 它催生了什么: 公式能说明训练目标，但不能单独说明世界状态持有；公式地图只负责阶段 IV 的外显生产闭合，不替代阶段 V 的状态审计。

## 一、公式卡片协议

阶段 IV 的公式以后按同一张卡读取：

```text
公式在问什么？
每个符号是什么？
哪个参数是旋钮？
旋钮变大 / 变小会怎样？
这个公式服务本章哪个闭合？
读不懂公式时，只需要记住哪一句话？
```

公式不是装饰。每个公式都必须说明自己服务哪一种闭合：

| 章节 | 闭合 | 公式角色 |
|---|---|---|
| `§31` | 可能性闭合 | 给“什么图像可能成立”一个概率或能量地形 |
| `§32` | 外显可信性闭合 | 让生成样本经受共同学习的判别边界 |
| `§33` | 内部语闭合 | 约束潜变量、码本或可逆坐标如何组织样本 |
| `§34` | 显形轨迹闭合 | 把生成写成加噪、去噪、速度场或一致映射 |
| `§35` | 意图接口闭合 | 让文本、结构或参考条件进入生成轨迹 |
| `§36` | 生产制度闭合 | 压缩、采样、蒸馏、控制和评估共同组织生产 |
| `§37` | 反身表征桥 | 中间状态开始被表征目标校准 |

## 二、`§31`：可能性地形

### 2.1 能量模型

```math
p(x)=\frac{\exp(-E(x))}{Z}
```

| 问项 | 回答 |
|---|---|
| 公式在问什么 | 一张图像 `x` 在模型看来有多可能。 |
| 符号 | `E(x)` 是能量，越低越可能；`Z` 是归一化常数。 |
| 关键旋钮 | 能量函数形状、采样步数、负样本质量。 |
| 变大 / 变小 | 能量差异越尖锐，分布越集中；采样不足会让模型只学到局部地形。 |
| 服务闭合 | 可能性闭合。 |
| 只记一句 | EBM 不是直接画图，而是给“像不像数据”规定一片地形。 |

### 2.2 Score

```math
s(x)=\nabla_x \log p(x)
```

| 问项 | 回答 |
|---|---|
| 公式在问什么 | 当前点应该往哪里移动，才更像数据。 |
| 符号 | `s(x)` 是 score，表示 log 密度对图像的方向导数。 |
| 关键旋钮 | 噪声尺度、score 网络容量、训练扰动范围。 |
| 变大 / 变小 | 噪声尺度大时学全局方向，小尺度时学细节方向。 |
| 服务闭合 | 可能性闭合到显形轨迹闭合的桥。 |
| 只记一句 | 绝对概率难算，方向更可用。 |

## 三、`§32`：GAN 与 WGAN

### 3.1 原始 GAN

```math
\min_G \max_D V(D,G)
=
\mathbb{E}_{x \sim p_{\text{data}}}[\log D(x)]
+
\mathbb{E}_{z \sim p_z}[\log(1-D(G(z)))].
```

| 问项 | 回答 |
|---|---|
| 公式在问什么 | 生成器 `G` 能否造出让判别器 `D` 难以区分的样本。 |
| 符号 | `x` 是真图，`z` 是噪声，`G(z)` 是假图，`D` 给出真实概率。 |
| 关键旋钮 | `D/G` 更新比例、学习率、判别器强度。 |
| 变大 / 变小 | `D` 太强，`G` 梯度枯竭；`D` 太弱，`G` 得不到有效反馈。 |
| 服务闭合 | 外显可信性闭合。 |
| 只记一句 | GAN = `G` 造图，`D` 挑错，样本可信性来自共同学习的边界。 |

### 3.2 Non-saturating GAN

```math
\min_G -\mathbb{E}_{z \sim p_z}[\log D(G(z))].
```

| 问项 | 回答 |
|---|---|
| 公式在问什么 | 当判别器太强时，如何让生成器仍然拿到可用梯度。 |
| 关键旋钮 | 判别器置信度、生成器学习率。 |
| 服务闭合 | 外显可信性的梯度修补。 |
| 只记一句 | 这不是换本体，而是换梯度条件。 |

### 3.3 WGAN

```math
W(p_r,p_g)
=
\inf_{\gamma \in \Pi(p_r,p_g)}
\mathbb{E}_{(x,y)\sim\gamma}[\|x-y\|].
```

| 问项 | 回答 |
|---|---|
| 公式在问什么 | 把生成分布搬到真实分布需要多少最小工作量。 |
| 符号 | `p_r` 真实分布，`p_g` 生成分布，`\gamma` 是搬运方案。 |
| 关键旋钮 | Lipschitz 约束、gradient penalty 权重。 |
| 变大 / 变小 | 约束太弱会不稳，太强会压制判别器表达。 |
| 服务闭合 | 外显可信性闭合的动力学稳定化。 |
| 只记一句 | WGAN 把“骗过判别器”改写成更连续的分布搬运距离。 |

## 四、`§33`：内部语

### 4.1 VAE ELBO

```math
\mathcal{L}_{\text{ELBO}}
=
\mathbb{E}_{q_\phi(z|x)}[\log p_\theta(x|z)]
-
D_{\text{KL}}(q_\phi(z|x)\,\|\,p(z)).
```

| 问项 | 回答 |
|---|---|
| 公式在问什么 | `z` 既要能重建图像，又不能偏离可采样先验太远。 |
| 符号 | `q_phi` 是编码器，`p_theta` 是解码器，`p(z)` 是先验，`KL` 是分布差异。 |
| 关键旋钮 | KL 权重、latent 维度、重建损失形式。 |
| 变大 / 变小 | 正则太强会发糊或 posterior collapse；太弱会潜空间不规整。 |
| 服务闭合 | 内部语闭合。 |
| 只记一句 | VAE 让生成系统有一个可采样、可压缩的连续内部坐标。 |

### 4.2 β-VAE

```math
\mathcal{L}_{\beta\text{-VAE}}
=
\mathbb{E}_{q_\phi(z|x)}[\log p_\theta(x|z)]
-
\beta D_{\text{KL}}(q_\phi(z|x)\,\|\,p(z)).
```

| 问项 | 回答 |
|---|---|
| 公式在问什么 | 用更强先验约束换更规整、更可能解耦的潜空间。 |
| 关键旋钮 | `β`。 |
| 变大 / 变小 | `β` 大，潜空间更规整但重建变差；`β` 小，重建更好但内部语更纠缠。 |
| 服务闭合 | 内部语闭合的可解释性压力。 |
| 只记一句 | β 是“重建质量”和“潜空间秩序”之间的旋钮。 |

### 4.3 VQ-VAE

```math
\mathcal{L}
=
\mathcal{L}_{\text{recon}}
+
\| \operatorname{sg}[z_e(x)] - e \|_2^2
+
\beta \| z_e(x) - \operatorname{sg}[e] \|_2^2.
```

| 问项 | 回答 |
|---|---|
| 公式在问什么 | 连续编码器输出如何被吸附到离散码本，并保持重建能力。 |
| 关键旋钮 | 码本大小、commitment 权重 `β`、重建损失。 |
| 变大 / 变小 | 码本太大难训练，太小信息瓶颈过强；commitment 太弱会码本漂移。 |
| 服务闭合 | 离散内部语闭合。 |
| 只记一句 | VQ-VAE 把图像内部语切成可重复调用的视觉词汇。 |

### 4.4 Flow change of variables

```math
\log p_X(x)
=
\log p_Z(f(x))
+
\log \left| \det \frac{\partial f(x)}{\partial x} \right|.
```

| 问项 | 回答 |
|---|---|
| 公式在问什么 | 可逆变换如何把复杂图像密度换算到简单潜变量密度。 |
| 关键旋钮 | 可逆层设计、Jacobian 计算方式、潜空间先验。 |
| 变大 / 变小 | 表达力越强越能拟合复杂分布，但计算和架构约束也更重。 |
| 服务闭合 | 严格内部坐标闭合。 |
| 只记一句 | Flow 要求内部语和样本之间可逆、可算、可密度化。 |

## 五、`§34`：扩散与显形轨迹

### 5.1 前向加噪

```math
q(x_t|x_0)
=
\mathcal{N}(\sqrt{\bar{\alpha}_t}x_0,(1-\bar{\alpha}_t)I).
```

| 问项 | 回答 |
|---|---|
| 公式在问什么 | 原图如何在时间步 `t` 被系统性加噪。 |
| 关键旋钮 | 噪声调度、时间步数。 |
| 变大 / 变小 | 噪声更大，结构更少；噪声更小，保留更多原图细节。 |
| 服务闭合 | 显形轨迹闭合的正向退显。 |
| 只记一句 | 前向过程把图像从可区分结构退回噪声。 |

### 5.2 DDPM 噪声预测

```math
\mathcal{L}_{\text{simple}}
=
\mathbb{E}_{t,x_0,\epsilon}
\left[
\|\epsilon-\epsilon_\theta(x_t,t)\|^2
\right].
```

| 问项 | 回答 |
|---|---|
| 公式在问什么 | 模型能否在每个时间步猜出当前图像里被加入的噪声。 |
| 符号 | `x_0` 原图，`x_t` 加噪图，`t` 时间步，`ε` 真噪声，`ε_theta` 预测噪声。 |
| 关键旋钮 | 时间步采样、噪声调度、网络容量。 |
| 变大 / 变小 | 大噪声训练全局结构，小噪声训练细节修复。 |
| 服务闭合 | 显形轨迹闭合。 |
| 只记一句 | 扩散最小读法：加噪、猜噪声、去噪。 |

### 5.3 CFG

```math
\hat{\epsilon}
=
\epsilon_\theta(x_t,\varnothing)
+
w\left(\epsilon_\theta(x_t,c)-\epsilon_\theta(x_t,\varnothing)\right).
```

| 问项 | 回答 |
|---|---|
| 公式在问什么 | 条件 `c` 要把无条件生成轨迹推多强。 |
| 关键旋钮 | guidance scale `w`。 |
| 变大 / 变小 | `w` 大，条件更强但多样性和自然度可能下降；`w` 小，更自由但更不听 prompt。 |
| 服务闭合 | 显形轨迹到意图接口的桥。 |
| 只记一句 | CFG 是条件增益器，不是语义理解证明。 |

### 5.4 Flow / velocity

```math
\mathcal{L}_{\text{FM}}
=
\mathbb{E}_{t,x_0,x_1}
\left[
\|v_\theta(x_t,t)-u_t(x_t)\|^2
\right].
```

| 问项 | 回答 |
|---|---|
| 公式在问什么 | 模型能否学习从噪声分布走向数据分布的速度场。 |
| 关键旋钮 | 路径构造、时间采样、速度参数化。 |
| 服务闭合 | 显形轨迹闭合的连续流版本。 |
| 只记一句 | Flow Matching 把去噪改写成沿速度场搬运。 |

Score-SDE、EDM、Consistency、Rectified Flow / Flow Matching 都属于二遍阅读对象。第一遍先抓 DDPM 与 CFG。

## 六、`§35`：意图接口

### 6.1 Cross-attention

```math
\operatorname{Attention}(Q_{\text{img}},K_{\text{txt}},V_{\text{txt}})
=
\operatorname{softmax}
\left(
\frac{Q_{\text{img}}K_{\text{txt}}^\top}{\sqrt{d}}
\right)
V_{\text{txt}}.
```

| 问项 | 回答 |
|---|---|
| 公式在问什么 | 图像生成的局部 token 如何读取文本 token。 |
| 关键旋钮 | attention 层位置、文本编码器、token 长度、条件 dropout。 |
| 变大 / 变小 | 注入太弱，prompt 不听；注入太强，图像被文本硬拉而结构不稳。 |
| 服务闭合 | 意图接口闭合。 |
| 只记一句 | 文本不是只给标签，而是逐层参与图像显形。 |

## 七、`§36`：生产制度

| 工艺公式 / 机制 | 它在问什么 | 最小读法 |
|---|---|---|
| LDM 压缩 | 为什么不直接在像素空间扩散 | 先压到 latent，再在 latent 里跑内部时间 |
| DPM-Solver / DPM++ | 少步采样如何保持质量 | 把采样看成数值积分问题 |
| Distillation | 如何把多步轨迹压成少步 | 用教师轨迹训练学生快速显形 |
| ControlNet | 如何接入硬结构条件 | 保留主干，同时增加结构控制支路 |
| LoRA / DreamBooth | 如何低成本吸收身份或风格 | 小参数更新换特定概念 |

本章的公式价值不在单一损失，而在生产制度栈：压缩、主干、条件、轨迹、采样、蒸馏、控制、个性化、数据、评估共同闭合。

## 八、`§37`：REPA 与反身表征桥

```math
\mathcal{L}
=
\mathcal{L}_{\text{diffusion}}
+
\lambda \mathcal{L}_{\text{align}}.
```

| 问项 | 回答 |
|---|---|
| 公式在问什么 | 生成训练是否应被外部强表征校准。 |
| 关键旋钮 | 对齐权重 `λ`、对齐层、外部表征器。 |
| 变大 / 变小 | `λ` 大，表征约束更强但可能压制生成自由；`λ` 小，退回普通生成训练。 |
| 服务闭合 | 反身表征桥。 |
| 只记一句 | REPA 不是证明 V 已完成，而是说明 IV 的生成动力开始需要 V 式表征校准。 |

## 九、整页最短读法

| 家族 | 一句话 |
|---|---|
| EBM | 给图像可能性画地形 |
| GAN | 让样本经受对手判别 |
| VAE | 让图像经过可采样潜坐标 |
| VQ-VAE | 把内部语切成离散词汇 |
| Flow | 让图像和潜变量可逆换算 |
| Diffusion | 用内部时间逐步显形 |
| CFG / cross-attention | 把意图推入生成轨迹 |
| REPA | 用强表征反过来校准生成 |
