# 阶段 IV 深化执行记录

日期: 2026-04-24  
范围: 阶段 IV 总纲、核心语法、§31-§37、导航、债务页、结语、索引与查看器清单  

## 本轮目标

把阶段 IV 从“生成模型发展史”进一步压成“外显生产共形”的理念展开史。

核心口径：

```text
(z, c, τ, D) -> x_hat
```

阶段 IV 的胜利是：视觉终于能把内部可能性外显为图像。  
阶段 IV 的失败是：外显为图像，不等于持有一个世界。

## 已执行

| 项目 | 处理 |
|---|---|
| 纯存在态 | 新增阶段 IV 核心语法页，定义 `(z,c,τ,D) -> x_hat` |
| 主体性边界 | 将早期完整主体口径收束为“外显生产主体性”，避免写成完整主体 |
| 闭合链 | 把 §31-§37 统一为可能性、可信性、内部语、显形轨迹、意图/工艺、反身表征六种闭合 |
| 评价账本 | 新增生成评价账本，拆分 likelihood、FID、CLIPScore、偏好、编辑一致性等度量残差 |
| §31 | 将“共同病理”改为前夜三类残差，并标清 DBN 的跨阶段读法 |
| §32 | 将 GAN 从早期强主体性表述改为“第一次外显可信性事件” |
| §33 | 增加内部语五判据，避免任何 latent 都被叫作内部语 |
| §34 | 明确扩散是“显形轨迹闭合”，不是简单价值排序意义上的高低判断 |
| §35 | 将文本条件改为“意图接口”，标明文本不是唯一条件接口 |
| §36 | 将最前沿清单改写为生产制度闭合，并补评价账本 |
| §37 | 将 REPA/iBOT/MAE 等节点压回 IV->V 桥接，不直接等同阶段 V 完成 |
