# 阶段 IV · 参数敏感度账本

类型: 理论/调参账本
阶段: IV · 生成共形
共形维度: 动力, 度量, 工艺
关键贡献: 把阶段 IV 常见“旋钮”从经验调参改写为闭合半径的敏感点。参数不是小数值，它们决定可信性、多样性、可控性、速度和内部语秩序之间如何交换。

## 一、为什么需要参数账本

阶段 IV 的很多失败不是“模型不会生成”，而是某个旋钮把闭合压偏了：

- 太听 prompt，图像僵硬或不自然。
- 太追求重建，潜空间失去秩序。
- 太追求先验，样本发糊。
- 太追求速度，结构崩坏。
- 太追求个性化，过拟合某个身份或风格。

因此，本页只问一件事：

> 参数变大 / 变小时，阶段 IV 的哪种闭合被强化，哪种闭合被牺牲？

## 二、核心敏感度表

| 模型 / 机制 | 关键旋钮 | 变大可能发生什么 | 变小可能发生什么 |
|---|---|---|---|
| GAN | 判别器强度、`D/G` 更新比 | 判别边界更锋利，但 `G` 可能梯度枯竭、训练振荡 | 反馈太弱，样本质量和可信性不足 |
| WGAN-GP | gradient penalty `λ` | 判别器更平滑，训练更稳；过大时表达被压制 | Lipschitz 约束弱，距离反馈不稳 |
| VAE | KL 权重、latent 维度 | 潜空间更规整、可采样；过强时重建发糊 | 重建更好，但潜空间散乱、采样差 |
| β-VAE | `β` | 更强解耦压力，更强内部语秩序；牺牲细节 | 更好重建，但可解释性和可控性变弱 |
| VQ-VAE | codebook size、commitment 权重 | 码本大，表达更丰富但更难利用；commitment 大，编码器更服从码本 | 码本小，信息瓶颈强；commitment 小，码本漂移或死码 |
| Flow | 可逆层深度、Jacobian 约束 | 表达力更强，但计算成本和架构约束更高 | 训练更轻，但复杂图像分布拟合不足 |
| DDPM | 时间步数、噪声调度 | 更多步更稳更细，但采样更慢 | 更快，但轨迹粗糙、质量下降 |
| Diffusion | 网络容量、噪声范围 | 覆盖更多尺度和细节；成本更高 | 成本低，但复杂结构和细节不足 |
| CFG | guidance scale `w` | prompt 跟随增强，但多样性、自然度和结构可能下降 | 更自然更多样，但更不听条件 |
| LDM | latent 压缩率 | 成本更低、速度更快；过强会损失细节 | 细节保留更多，但计算成本上升 |
| ControlNet | 控制权重、控制层位置 | 更服从边缘、姿态、深度等结构；过强会僵硬 | 更自由，但结构约束容易失效 |
| LoRA / DreamBooth | rank、训练步数、学习率 | 更强个性化；过大过久会过拟合、污染基模型风格 | 更稳但身份或风格注入不足 |
| DiT | patch size、模型深度、token 数 | 更强全局建模和 scaling；成本上升 | 更高效，但全局关系和细节可能不足 |
| 蒸馏 / LCM | 蒸馏步数、teacher 强度 | 更快，少步质量更好；过强可能学到 teacher 偏差 | 更接近原轨迹，但速度收益不足 |
| 采样器 | step 数、solver 阶数、噪声 schedule | 质量和稳定性提升；时间成本上升 | 更快，但细节、结构和 prompt follow 更容易崩 |
| REPA | 对齐权重 `λ` | 中间状态更接近强表征；过强可能压制生成自由 | 生成更自由，但表征校准弱 |

## 三、按闭合类型看旋钮

| 闭合类型 | 最敏感旋钮 | 常见交换 |
|---|---|---|
| 可信性闭合 | 判别器强度、WGAN-GP `λ` | 锐利性 vs 稳定性 |
| 内部语闭合 | KL、`β`、codebook size、latent 压缩率 | 可采样秩序 vs 重建细节 |
| 显形轨迹闭合 | 时间步数、noise schedule、solver 阶数 | 质量稳定 vs 速度成本 |
| 意图接口闭合 | CFG scale、cross-attention 注入、ControlNet 权重 | 条件服从 vs 自然多样 |
| 生产制度闭合 | 蒸馏强度、LoRA rank、数据过滤阈值 | 部署效率 vs 泛化与可靠性 |
| 反身表征桥 | REPA `λ`、对齐层 | 表征校准 vs 生成自由 |

## 四、最短使用协议

读阶段 IV 的任何参数时，先问四个问题：

1. 它强化了哪一种闭合？
2. 它牺牲了哪一种闭合？
3. 它改变的是样本、轨迹、条件、内部语，还是评价？
4. 它的失败会把路线推向 §35、§36、§37 还是阶段 V？

如果一个参数只被写成“调大效果更好”，它还没有进入本大厦的理论语言。
