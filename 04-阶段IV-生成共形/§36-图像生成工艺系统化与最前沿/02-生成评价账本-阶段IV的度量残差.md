# 阶段 IV · §36-A：生成评价账本——阶段 IV 的度量残差

类型: 理论/横切账本  
阶段: IV · 生成共形  
共形维度: 度量, 反身, 动力  
关键贡献: 把阶段 IV 的评价从“画得好不好”拆成多种度量对象，说明每一种指标命中了什么、遮蔽了什么。  
残差 / 它催生了什么: 生成样本越强，评价越不透明；这条度量残差把阶段 IV 接到反身脊柱，并为阶段 V 的状态持有判据铺路。  

## 一、为什么阶段 IV 必须有评价账本

阶段 I 的度量可以压到 Top-1 / Top-5，阶段 II 可以压到 IoU / mAP / PQ，阶段 III 可以压到图文对齐、VQA、grounding。但阶段 IV 不行。

生成评价天然是多目标的：

```text
likelihood
FID
IS
CLIPScore
aesthetic score
human preference
prompt alignment
edit consistency
identity preservation
structural consistency
diversity
speed
cost
safety
```

这些指标不是同一件事的不同名字，而是在裁判不同对象。若把它们混成一个“生成质量”，阶段 IV 的理论会被压扁。

## 二、主要度量命中什么

| 度量 | 命中的对象 | 遮蔽的对象 |
|---|---|---|
| likelihood | 分布概率与显式密度 | 感知质量、语义对齐、样本锐度 |
| FID | 生成集与真实集的特征分布相似性 | 个体语义正确性、prompt follow、关系一致性 |
| IS | 类别可判别性与多样性的一种近似 | 开放语义、空间结构、任务可控性 |
| CLIPScore | 图文整体相似 | 计数、左右、空间关系、遮挡、局部错误 |
| aesthetic score | 人类偏好中的审美面 | 世界真实性、任务约束、细节可靠性 |
| human preference | 观者偏好与产品体验 | 长期一致性、真实状态、不可见错误 |
| prompt alignment | 文本意图是否被响应 | 未写入 prompt 的不变量、物理与几何约束 |
| edit consistency | 局部改动是否守约束 | 全局世界状态、非编辑区域的隐性漂移 |
| identity preservation | 主体身份是否保持 | 背景、关系、姿态、其他对象一致性 |
| diversity | 分布覆盖与样本变化 | 单个样本质量、意图遵循 |
| speed / cost | 生产效率 | 理解质量、结构可靠性 |
| safety filters | 输出合规 | 世界真实性、细粒度理解 |

## 三、阶段 IV 的三类度量残差

### 3.1 好看不等于成立

高审美、低 FID 或高偏好并不保证：

- 手指数量正确。
- 文字可读。
- 空间关系成立。
- 多主体关系稳定。
- 场景具有可解释世界状态。

这是样本可信性闭合的残差。

### 3.2 对齐不等于持有

CLIPScore、prompt alignment 或人类偏好可以说明模型“响应了文本”，但不能证明系统在内部持有对象、几何、遮挡、身份和因果状态。

这是意图接口闭合的残差。

### 3.3 可编辑不等于可守恒

编辑指标容易只看“改了该改的地方”，但真正困难的是：

```text
什么不该改？
```

只要身份、背景、光照、空间布局或其他对象在局部编辑中漂移，说明系统还没有稳定持有不变量。

这是编辑控制闭合的残差，也是阶段 IV 通往阶段 V 的关键入口。

## 四、反身结论

阶段 IV 的度量危机可以压成一句话：

> 生成样本越强，评价越不透明。

因此，任何声称“生成模型理解了世界”的判断，都必须说明：

1. 用什么度量判定闭合？
2. 这个度量命中了什么对象？
3. 这个度量遮蔽了什么对象？
4. 被遮蔽对象如何以残差返回？
5. 下一阶段是否改写了度量本身？

这正是反身脊柱在阶段 IV 的具体形态。

## 五、不同问题应该选什么指标

| 评估问题 | 不要只看 | 更应该看 |
|---|---|---|
| 生成集像不像真实分布 | 单张样本美不美 | FID / precision-recall / diversity |
| prompt 是否被执行 | FID 或审美分 | prompt alignment、人工偏好、对象计数与关系测试 |
| 空间关系是否成立 | CLIPScore | 关系、左右、上下、遮挡、布局一致性测试 |
| 编辑是否守住不变量 | 改动区域是否成功 | 非编辑区域一致性、identity preservation、背景漂移 |
| 主体身份是否保持 | 人类整体偏好 | face / object identity、跨姿态一致性 |
| 是否理解世界状态 | 画面是否逼真 | 几何、遮挡、对应、状态预测与反事实一致性 |

这张表的结论是：阶段 IV 的度量不能压成一个 leaderboard 分数。不同指标命中不同对象，误用指标会制造伪闭合。
