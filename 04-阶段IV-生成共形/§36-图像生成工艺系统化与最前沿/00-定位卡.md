# 阶段 IV · §36 定位卡：图像生成工艺系统化与最前沿

类型: 理论/定位卡
阶段: IV · 生成共形
共形维度: 时间尺度, 动力, 材料
关键贡献: 把 diffusion 在图像生成上的后半程写成一条可系统化的工艺主线：从 LDM / SDXL 到 DiT / MMDiT / SD3 / Flux / Rectified Flow / Consistency / distillation / editing-control，阶段 IV 在这里真正把“图像生成”吃到最前沿。
残差 / 它催生了什么: 当图像生成的主工艺链已经系统化，真正的残差就不再是“还能不能更会画”，而是“中间状态是否比最终图像更像理解”。于是 `§37` 开始把生成器倒转成表征器。
定位标签: diffusion frontier, DiT, MMDiT, SD3, Flux, rectified flow, consistency

## 本章位置

这一章说明：阶段 IV 的成熟形态不是“已经会画图”就结束，而是把 diffusion 从范式革命推进到完整工艺系统。

## 代表线索

- 主干架构：LDM / SDXL / DiT / MMDiT / SD3 / Flux
- 动力学再设计：EDM / Consistency / Rectified Flow / distillation
- 图像工艺生态：ControlNet / IP-Adapter / inpainting / editing / personalization

## 向下一章的传递

当图像生成已经高度成熟，阶段 IV 必须开始反问：真正重要的是最终图像，还是生成过程中形成的内部状态？这条追问送向 `§37`。
