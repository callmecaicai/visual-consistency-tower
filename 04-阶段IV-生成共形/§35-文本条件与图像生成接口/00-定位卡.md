# 阶段 IV · §35 定位卡：文本条件与图像生成接口

类型: 理论/定位卡
阶段: IV · 生成共形
共形维度: 载体, 主体性, 动力
关键贡献: 把 DALL·E / Imagen / Parti / SDXL 以及 prompt / adapter / guidance 系统定位为阶段 IV 的意图接口工程：文本成为最通用、最公共、最容易规模化的人类意图载体，结构条件继续补足“必须怎样生成”的约束。
残差 / 它催生了什么: 文本与结构条件让外显生产具备可控性，但仅有接口不等于已经把 diffusion 工艺系统化，于是进入 `§36` 的生产制度主线。
定位标签: text-to-image, DALL-E, Imagen, SDXL, prompt, adapter

## 本章位置

本章不是把阶段 IV 拉回阶段 III，而是说明：当外显生产需要一个通用意图接口时，文本是最顺手的公共载体，但不是唯一条件接口。

## 代表线索

- 文本到图像：DALL·E / Imagen / Parti / SDXL
- prompt / guidance / adapter：让文本条件真正可用
- 关键判断：LLM 或大语言接口在此只是条件供给者，不是阶段命名者

## 向下一章的传递

图像生成已经能受文本和结构条件约束，但 diffusion 的训练、架构、采样、控制、蒸馏、评价和前沿工艺仍未被系统化。`§36` 要把这条线真正吃透。
